/add



Note: This feature has essentially been replaced by /load.

The /add command could previously be used to replace the currently loaded aliases, popups, commands, remote users, and events sections from the specified ini file. The ini file must be formatted the same way /save formats them.

Synopsis

/add [-apuce] <file.ini>

Switches

-a - Load the aliases
-p - Load the popups
-u - Load the remote users
-c - Load the commands
-e - Load the events

Parameters

<file.ini> - The initialization file to load the appropriate section from.

Example

Prints:
*** Added: Commands from c:\mirc\helpfulCmds.ini 
/add -a helpfulCmds.ini

Compatibility

Added: mIRC v3.3 (21 Jun 1995)
Removed: mIRC v3.8 (25 Nov 1995)

Notes

None
