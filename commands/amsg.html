/amsg



The /amsg command is used to send a message to all the channels you are currently on. To send a message to a specific channel, consider using the /msg command instead.

Synopsis

/amsg <message>

Switches

None

Parameters

<message> - The message to be sent.

Example

/amsg It's snowing!

Compatibility

Added: mIRC v3.2 (02 May 1995)

Notes

As of mIRC v7.1 the /amsg command will split long messages into smaller messages so no part of the message will be truncated.
