/autojoin



The /autojoin command is used in conjunction with the on Connect event (or the Perform option available in alt + o > options > perform..) in order to delay or prevent auto-joining of channels. This also apply to rejoining of open channels windows during reconnect

Synopsis

/autojoin -nsdN

Switches

-n - Join now
-s - Skip autojoin
-dN - Delay auto-join for N seconds

Parameters

None

Example

on *:connect:{
  /*
  5 seconds delay before we re-join the channels
  Enough time to perform more important things
  */
  autojoin -d5

  ns id FooBar
  mode $me -x
}

Compatibility

Added: mIRC v6.17 (17 Feb 2006)

Notes

The max delay is 600 seconds, anything more resets to 600.
