/closemsg



Note: commands been superseded by the /close
The /closemsg command can be used to close all query windows. A name can be specified to close only that query window.

Synopsis

/closemsg [nick]

Switches

None

Parameters


[nick] - a nickname for the window to close

Example

; Close Kol's query
/closemsg kol

Compatibility

Added: mIRC v3.3 (21 Jun 1995)

Notes

None
