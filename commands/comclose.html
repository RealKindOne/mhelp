/comclose



The /comclose command closes an existing :doc: `COM connection TODO_LINK_ADVANCED_COM` by a specific handle. The supplied com handle must have been previously created via the /comopen or the $com / $comcall identifiers.

Synopsis

/comclose <name>

Switches

None

Parameters


<name> - The name of the connection to close

Example

Alias Example {
  ; Create a WshShell object
  comopen Example wscript.shell
  ; Destroy object
  comclose Example
}

Compatibility

Added: mIRC v5.9 (15 Jun 2001)
