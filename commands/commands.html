/commands



Note: This feature has essentially been replaced by /ctcps.

The /commands command turns CTCP event processing on or off.

Synopsis

/commands [on|off]

Switches

None

Parameters


<on|off> - turns CTCP event processing on or off

Example

; disable ctcps
/commands off

Compatibility

Added: mIRC v4.0 (20 Mar 1996)
Removed: mIRC v5.0 (21 Apr 1997)

Notes

None
