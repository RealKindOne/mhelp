/ctcp



The /ctcp command can be used to send a CTCP request to a user/channel

Synopsis

/ctcp <nick/#channel> <ctcp>

Switches

None

Parameters

<nick> - Target nick/channel
<nick> - CTCP

Example

; Version everyone in #mSL
/ctcp #mSL VERSION

Compatibility

Added: mIRC v2.1a (28 Feb 1995)

Notes

None
