/ctcps



The /ctcps command is used to turn processing of CTCP events on/off. With no parameters, this command will display whether or not CTCPs are on or off.

Synopsis

/ctcps [on|off]

Switches

None

Parameters

on - Enable CTCP events
off - Disable CTCP events

Example

Display ctcps status: 
/ctcps
Enable ctcps: 
/ctcps on
Disable ctcps: 
/ctcps off

Compatibility

Added: mIRC v5.0 (21 Apr 1997)
