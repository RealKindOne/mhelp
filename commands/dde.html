/dde



Dynamic Data Exchange (DDE) is a form of communication between two applications on a windows machine.
The /dde command can be used to create a dde connection with a dde server. When the topic parameter is required, you can use empty quotes ("") to indicate a blank topic.
Transactions The dde command support three different types of transactions:

XTYP_POKE - Used by default, sends unsolicited data to the server.
XTYP_REQUEST - Used to request data from a server.
XTYP_EXECUTE - Used to send a command string to the server. The only time the [data] parameter is required is when the XTYP_POKE transaction type is used.

Synopsis

/dde [-re] <service> <topic> <item> [data]

Switches


-r - uses XTYP_REQUEST transaction
-e - uses XTYP_EXECUTE transaction

Parameters


<service> - DDE Server service name
<topic> - DDE Topic
<item> - DDE Item
[data] - Data to be sent

Example

; Open a new tab in opera, navigate to www.google.com
/dde -e Opera WWW_OpenURL "www.google.com"

Compatibility

Added: mIRC v3.9 (06 Jan 1996)
