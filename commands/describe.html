/describe



The /describe command sends an action message to a specific channel or person. This command is the same the /me except you can set a destination channel or nick other then the active window.

Synopsis

/describe <nick|channel>

Switches

None

Parameters

<nick|channel> - The destination channel or person for the action message
<message> - The action message to be sent

Example

; When someone says 'moo', we will reply
on *:Text:moo:#:{
  describe $chan moos back!
}

Compatibility

Added: mIRC v3.1 (23 Apr 1995)
