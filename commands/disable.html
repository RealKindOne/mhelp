/disable



The /disable command disables the specified groups in all scripts. You can also specify a wildcard to disable all matching groups.

Synopsis

/disable <group1 group2 ... groupN>
/disable <wilcard_expression>

Switches

None

Parameters

<group1 group2 ... groupN> - A list with complete name of the groups to be disabled separated by spaces.
<wildcard_expression> - A wildcard expression, all matching groups are disabled.

Example

Disable the three groups specified: 
/disable #one #two #three
Disable the first two (#one and #two, assuming that's the only matching groups): 
/disable #*o*

Compatibility

Added: mIRC v3.5 (13 Aug 1995)

Notes
