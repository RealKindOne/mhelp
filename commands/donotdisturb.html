/donotdisturb



The /donotdisturb command allows you to temporarily disable all visible/audible notifications such as sounds, tips, flashing window icons, tray icon animations, and more.

Synopsis

/donotdisturb [on|off]

Switches

None

Parameters

on - Enable donotdisturb feature
off - Disable donotdisturb feature

Example

/donotdisturb on

Compatibility

Added: mIRC v7.62 (29 Jul 2020)

Notes

Only works on Windows Vista and newer. If you attempt to load this on Windows XP it will disable itself.
