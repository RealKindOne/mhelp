/drawdll



The /drawdll command that allows a script to pass the active memory HDC and HBITMAP handle for a @window to a custom DLL, which can then draw directly to it.

Synopsis

/drawdll -n <@window> <dllname> <procname> [data]

Example

Draws a small black dot in the top left of the window: 
alias test {
  window -p @test 100 100 200 200
  drawdll @test M:\mIRC\DLL\dll.dll draw
}
