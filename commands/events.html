/events



The /events command can be used to toggle the remote events processing. With no parameters, the /events command displays the current state (either disabled or enabled).

Synopsis

/events [on|off]

Switches

None

Parameters

on - Enable event processing.
off - Disable event processing.

Example

Display events status: 
/events
Enable events: 
/events on
Disable events: 
/events off

Compatibility

Added: mIRC v4.0 (20 Mar 1996)
