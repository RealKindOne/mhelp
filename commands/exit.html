/exit



The /exit command can be used to safely close and exit mIRC.

Synopsis

/exit -nr

Switches

-n - Switch disables the exit confirmation dialog.
-r - Switch restarts mIRC.

Parameters

None

Example

Restart mIRC: 
/exit -r

Compatibility

Added: mIRC v3.3 (21 Jun 1995)

Notes

As of version 7.11 /exit -r will also pass the -i, -r, -noreg, and -portable command line parameters to the new mIRC.
