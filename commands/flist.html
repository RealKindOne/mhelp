/flist



The /flist command lists all open files, or those matching the specified name, which can be a wildcard expression.

Synopsis

/flist [name]

Switches

None

Parameters


[name1] - a wildcard expression

Example



    /flist
    /flist myhandle*

Compatibility

Added: mIRC v6.1 (17 Feb 2006)

Notes

None
