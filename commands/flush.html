/flush



The /flush command clears the remote user list of nickname definitions that are no longer valid. For each nick in the remote user list that matches the specified levels mIRC checks to see if that nick is on any of the channels that you are currently on. If not, the nick definition is removed from the remote user list. If you do not specify levels then mIRC clears all nicks from the remote user list that do not exist on channels you are on.

Synopsis

/flush -l [levels]

Switches


-l - removes only the specified levels from entries in the user list, instead of removing the entries

Parameters


[levels] - A list of levels seperated by commas

Example



    /flush 1,2,3

Compatibility

Added: mIRC vmIRC 3.8 ()
