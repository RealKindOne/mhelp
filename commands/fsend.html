/fsend

/fsend



The /fsend command turns on/off the fast DCC send feature. Displays a line with the current status (active window if applicable, status window otherwise).

Synopsis

/fsend [on|off|N]

Switches

None

Parameters

on - Enable fast DCC send.
off - Disable fast DCC send.
[N] - A number between 0 and 100, experimental fast DCC feature.

Example

Display fsend status: 
/fsend
Enable fsend: 
/fsend on
Disable fsend: 
/fsend off

Compatibility

Added: mIRC v3.3, v3.4 (21 Jun 1995)
