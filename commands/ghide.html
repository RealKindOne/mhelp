/ghide

/ghide



The ghide command hides a previously loaded agent.

Synopsis

/ghide -h <name>

Switches

-h - Prevents the agent from poping up if mIRC is minimized and -h has been used in /gload to load that agent

Parameters

<name> - The name under which you loaded the agent

Example

Load the default agent on your system: 
/gload myagent default
Hide the agent: 
/ghide myagent

Compatibility

Added: mIRC v5.7 (07 May 2000)

Notes

None
