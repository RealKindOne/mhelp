/gmove



The /gmove command moves an agent.

Synopsis

/gmove -h <name> >x> >y> [speed]

Switches

-h - Prevents the agent from poping up if mIRC is minimized and -h has been used in /gload to load that agent

Parameters

<name> - The name used in /gload to reference that agent
<x> - The 'x' coordinate
<y> - The 'y' coordinate
[speed] - If a speed is not specified, it uses a default speed. If you specify a speed of 0, it moves instantly to the new position.

Example

Load the default agent on your system: 
/gload myagent default
Move the agent to coordinate 42,42 with a speed of 2: 
/gmove myagent 42 42 2

Compatibility

Added: mIRC v5.7 (07 May 2000)

None
