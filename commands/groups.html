/groups

/groups



The /groups command displays a list of either all (no switch), enabled (-e), or disabled (-d) groups in your scripts.

Synopsis

/groups [-e|d]

Switches

-e - Only enabled groups are displayed.
-d - Only disabled groups are displayed.

Parameters

None

Example

Display all groups status: 
/groups
Display only enabled groups: 
/groups -e
Display only disabled groups: 
/groups -d

Compatibility

Added: mIRC v4.0 (20 Mar 1996)

Notes

None
