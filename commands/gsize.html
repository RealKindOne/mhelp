/gsize



The /gsize command resizes an agent to the specified width and height.

Synopsis

/gsize -h <name> <w> <h>

Switches


-h - Prevents the agent from poping up if mIRC is minimized and -h has been used in /gload to load that agent

Parameters


<name> - The name used in /gload to reference that agent
<w> - The new width of the agent
<h> - The new height of the agent

Example

; Load the default agent on your system
/gload myagent default
; Resize the agent to a width of 42 and a height of 42
/gsize myagent 42 42

Compatibility

Added: mIRC v5.7 (07 May 2000)

Notes

None
