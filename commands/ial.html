/ial

/ial



The /ial command turns IAL (Internal Address List) on or off. If no on/off parameter is used, it will display the IAL status

Synopsis

/ial [on|off]

Switches

None

Parameters

on - Enable the internal address list.
off - Disable the internal address list.

Example

Display IAL status: 
/ial
Enable IAL: 
/ial on
Disable IAL: 
/ial off

Compatibility

Added: mIRC v4.7 (09 Jan 1997)

Notes

This setting is not persistent across sessions and resets to on every time mIRC is run.
