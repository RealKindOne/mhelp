/ialclear

/ialclear



The /ialclear command clears the IAL (Internal Address List).

Synopsis

/ialclear [nickname]

Switches

None

Parameters

[nickname] - If specified, the IAL is only cleared for that nickname, otherwise the whole list is cleared

Example

Clear the entire IAL: 
/ialclear
Clear Bob from the IAL: 
/ialclear Bob

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

None
