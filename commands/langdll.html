/langdll



The /langdll command is used to create a default language DLL that contains translatable strings.

Synopsis

/langdll -me[d] [oldmirc.exe] [newmirc.exe]

Switches

-m - Create the DLL
-e - List supported language codes
-d - Compare two versions of mIRC

Parameters

[code] - Used with -m switch. Can be "0x0409", "1033", or "en-US".
[oldmirc.exe] - Old mIRC version to compare against.
[newmirc.exe] - New mIRC version to compare against.

Example

Create en-US DLL: 
/langdll -m 1033
Compare mIRC Versions: 
/langdll -d oldmirc.exe newmirc.exe

Compatibility

Added: mIRC v7.77 (11 Jun 2024)
