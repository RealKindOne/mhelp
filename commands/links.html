/links



The /links command can be used to retrieve the servers to which your current server is linked which will be displayed in a window.

Synopsis

/links -nx

Switches


-n - Minimize the window
-x - Maximize the window

Parameters

None

Example

None

Compatibility

Added: mIRC v4.1 (27 Apr 1996)

Notes

None
