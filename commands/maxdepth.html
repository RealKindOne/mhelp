/maxdepth



Note: This feature has been removed.

The /maxdepth command could be used to change the maximum allowable recursion limit for a single alias. A depth of 1 indicates no direct recursion is allowed. The maximum depth is 100. Calling an alias from within itself recursively has been disabled.

Synopsis

/maxdepth <depth>

Switches

Parameters

<depth> - The new recursion depth limit (1-100).

Example

example {
  ;set max recursion depth to 3
  maxdepth 3
  set %a 1
  test
}
test {
  echo test %a
  inc %a
  test
}
Output: 
   * Max depth is 3
   -
   test 1
   test 2
   -
   TEST Unknown command
   -
Another Example: 
example {
  maxdepth 10
  set %a 1
  test
}
test {
  echo test %a
  inc %a
  test
}
Output: 
   -
   * Max depth is 10
   -
   test 1
   test 2
   test 3
   test 4
   test 5
   test 6
   test 7
   test 8
   test 9
   -
   TEST Unknown command
   -
