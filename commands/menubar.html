/menubar



The /menubar command display the current status and optionally turns the menubar on/off.

Synopsis

/menubar <on|off>

Switches

None

Parameters

on - Enable the mIRC menubar.
off - Disable the mIRC menubar.

Example

Display menubar status: 
/menubar
Enable the menubar: 
/menubar on
Disable the menubar: 
/menubar off

Compatibility

Added: mIRC v6.32 (21 May 2008)

Notes

While the menubar is off, the menubar's options are now accessed from left-clicking the mIRC icon in the upper left corner of the main window, or right-clicking anywhere in the main window's titlebar.
