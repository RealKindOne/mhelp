/pdcc



The /pdcc command attempts to speed up the mIRC DCC send feature by sending packets ahead of acks, or acknowledgements.

Synopsis

/pdcc [on|off]

Switches

None

Parameters

on - Enable DCC pumping.
off - Disable DCC pumping.

Example

Display pdcc status: 
/pdcc
Enable pdcc: 
/pdcc on
Disable pdcc: 
/pdcc off

Compatibility

Added: mIRC v3.3, v3.4 (21 Jun 1995)
