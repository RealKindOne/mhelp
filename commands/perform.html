/perform



The /perform command enables or disables the perform section in the options (ALT + O > Connect)

Synopsis

/perform [on | off]

Switches

None

Parameters


[on|off] - If specified, turns the feature on/off, otherwise displays the current state of the feature

Example

None

Added: mIRC v5.8 (14 Dec 2000)
