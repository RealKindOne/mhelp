/playctrl



The /playctrl command opens the Play Central window, which can be used to manage the queued /play requests.

Synopsis

/playctrl

Switches

None

Parameters

None

Example



    ;Open the play central
    /playctrl

Compatibility

Added: mIRC v5.81 (09 Nov 2000)

Notes

None
