/pop



The /pop command performs a delayed op on a nickname. The purpose of this command is to prevent a channel window filling up with op mode changes whenever several users have the same nickname in their auto-op section. Before performing the op it checks if the user is already opped.

Synopsis

/pop <delay> [#channel] <nickname>

Switches

None

Parameters

<delay> - The number of seconds (1 - 15) before performing the op. If <delay> is not used the voice is immediate.
[#channel] - If specified, the channel you want to op the nickname on, otherwise the current channel is used.
<nickname> - The nickname you want to op.
