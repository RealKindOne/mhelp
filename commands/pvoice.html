/pvoice



The /pvoice command performs a delayed voice on a nickname. The purpose of this command is to prevent a channel window filling up with voice mode changes whenever several users have the same nickname in their auto-voice section. Before performing the voice it checks if the user is already voiced.

Synopsis

/pvoice <delay> [#channel] <nickname>

Switches

None

Parameters

<delay> - The number of seconds (1 - 15) before performing the voice. If <delay> is not used the voice is immediate.
[#channel] - If specified, the channel you want to voice the nickname on, otherwise the current channel is used.
<nickname> - The nickname you want to voice.

Example
