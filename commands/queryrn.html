/queryrn



The /queryn command changes the nickname of an open query window.

Synopsis

/queryn <oldname> <newname>

Switches

None

Parameters

<oldname> - The old name of the query window.
<newname> - The new name of the query window.

Example

/requeryn Bob_ Bob

Compatibility

Added: mIRC v5.8 (14 Dec 2000)

Notes

None
