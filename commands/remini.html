/remini



The /remini command deletes a whole section or a single item in an INI file
Note: you shouldn't be using this on the INI files used by mIRC but that should work.

Synopsis

/remini  
 [item]

Switches

None

Parameters

* - - The filename of the ini file * -
- The section of the ini file you want to delete * - [item] - If speficied, the item you want to delete in the section, otherwise the whole section is deleted
