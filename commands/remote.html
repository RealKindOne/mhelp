/remote

/remote



The /remote command toggles the remote, CTCPS, and raw events processing. With no parameters, the /remote command displays the current state (either disabled or enabled).

Synopsis

/remote [on|off]

Switches

None

Parameters

on - Enable the event processing
off - Disable the event processing

Example

Display remote status: 
/remote
Enable remote: 
/remote on
Disable remote: 
/remote off

Compatibility

Added: mIRC v4.0 (20 Mar 1996)

Notes
