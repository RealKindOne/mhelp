/renwin



The /renwin command renames an existing window and assigns an optional topic.

Synopsis

/renwin <@currentName> <@newName> [topic]

Switches

None

Parameters

* - <@currentName> - The current window name * - <@newName> - The new window name * - [topic] - Window's topic

Example



    ;rename @foo to @bar
    /renwin @foo @bar

Compatibility

Added: mIRC v5.4 (24 Jul 1998)

Notes

None
