/resetidle



The /resetidle command resets the $idle identifier to zero or to the number of seconds you specify.

Synopsis

/resetidle [seconds]

Switches

None

Parameters

[seconds] - If specified, reset the $idle identifier to that number of seconds.

Example

/resetidle 5

Compatibility

Added: mIRC v5.3 (13 Dec 1997)

Notes

None
