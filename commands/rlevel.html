/rlevel



The /rlevel command removes all users from the remote users list with the specified general access level

Synopsis

/rlevel [-r] <levels>

Switches

-r - Removes the specified levels for that users and not just the general access level, if the user has no level, the entry is removed from the users list.

Parameters

<levels> - A comma seperated list of level to be removed.

Example

None
