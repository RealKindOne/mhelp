/rmdir



The /rmdir command deletes an empty directory. If the directly contains either files or other directories, an error will occur.

Synopsis

/rmdir

Switches

None

Parameters

* - - The path/name of the directory to delete

Example



    ; Create a directory
    /mkdir example

    ; Delete it
    /rmdir example

Compatibility

Added: mIRC v5.3 (04 Jan 1998)

None
