/say



The /say command sends a message to the active window which can be a channel or a query. The say command can only be used within events that can be triggered directly by you (i.e. on input, on tab completion, on hotlink, etc...). To send a message from any event, use the /msg command.

Synopsis

/say <message>

Switches

None

Parameters

<message> - The actual message to be sent.

Example

/say Hello.
