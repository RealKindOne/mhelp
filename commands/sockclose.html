/sockclose

/sockclose



The /sockclose command allows the closing of any open socket connections, either specifically or by wildcard.

Synopsis

/sockclose <name>

Switches

None

Parameters

<name> - the specific socket name to be closed. This parameter can also be a wildcard match.

Example

/sockclose mySock

Compatibility

Added: mIRC v5.3 (04 Jan 1998)

Notes

If you close a socket inside a socket event, all information on the socket are lost and $sockname will be set to $null as well.
