/tips



The /tips command can be used to disable and enable all tips options from the option dialog. Without any parameters, it prints the current tips settings. Turning tips off will kill all active tips, including the ones generated by :doc: `$tip ` .

Synopsis

/tips [on|off]

Switches

None

Parameters


[on|off] - on/off settings

Example



    ; disable all tips
    /tips off

Compatibility

Added: mIRC vmIRC 6.31 ()

Notes

None
