/tnick



The /tnick command can be used to change own IRC nickname without changing the main or alternative nick /anick options.

Synopsis

/tnick <newName>

Switches

None

Parameters

<newName> - The temporary nickname.

Example

alias away {
  ; change nick
  tnick $me $+ |away
  !away
}

Compatibility

Added: mIRC 6.0 (03 Feb 2002)

Notes

None

See also

$nick
$anick
