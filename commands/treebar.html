/treebar



The /treebar command is used to enable or disable the mIRC treebar. You can display the treebar status without including a parameter.

Synopsis

/treebar [on|off]

Switches

None

Parameters

on - Enable the treebar.
off - Disable the treebar.

Example

Display treebar status: 
/treebar
Enable treebar: 
/treebar on
Disable treebar: 
/treebar off

Compatibility

Added: mIRC v6.2 (23 Nov 2006)

Notes

The treebar cannot be disabled while in Compact mode.
