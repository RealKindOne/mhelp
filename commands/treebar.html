/treebar



The /treebar command is used to enable or disable the mIRC treebar.

Synopsis

/treebar [on|off]

Switches

None

Parameters


[on] - Enable the mIRC treebar.
[off] - Disable the mIRC treebar.

Example

; Enable the treebar
/treebar on
; Disable the treebar
/treebar off
; View the treebar status**
/treebar
The above command will result in the following: [[File:Treebar-Status.png]]

Compatibility

Added: mIRC v6.2 (23 Nov 2006)

Notes

The treebar cannot be disabled while in Compact mode.
