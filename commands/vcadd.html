/vcadd



The /vcadd command makes it simple for you to add a voice command to the command list for your Speech Recognition Software.

Synopsis

/vcadd <command1,command2,...&;gt;

Switches

None

Parameters

None

Example

Add a voice command for connecting to the SwiftIRC IRC server: 
/vcadd Connect SwiftIRC

Compatibility

Added: mIRC v5.7 (07 May 2000)

Notes

None
