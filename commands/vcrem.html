/vcrem



The /vcrem command can be used to remove a command from the voice command command list.

Synopsis

/vcrem -ra <command1>[,command2[,etc..]]

Switches


-r - removes the entry (default)
-a - adds the entry

Parameters


<command1> - command to remove

Example

None

Compatibility

Added: mIRC v5.7 (07 May 2000)

Notes

None
