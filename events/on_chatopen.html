On ChatOpen



The on CHATOPEN event triggers within mIRC when a DCC CHAT session starts.

Note: This feature is depreciated and has been replaced with On_Open.

Synopsis

on <level>:CHATOPEN:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Local Identifiers

=$nick - Refer to the DCC chat window, rather than a query window.

Example

on 1:CHATOPEN:msg =$nick Hi! I'll be with you in a second...

Compatibility

Added: mIRC v4.52 (06 Jul 1996)

Notes

Depreciated mIRC v5.3 (13 Dec 1997)
