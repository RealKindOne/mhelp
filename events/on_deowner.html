On DeOwner



The ON DEOWNER event triggers whenever a user on a channel has been released of their owner (mode -q) status.

Synopsis

ON <level>:DEOWNER:<#[,#]>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<#[,#]> - The place, or places where the event listens, you can specify specific name of window, separate them by comma.
# - Any channel window.
#chan - Only in #chan window.
#chan,#chan2 - Only in #chan and #chan2 window.
<commands> - The commands to be performed when the event triggers.

Local Identifiers

$opnick - Returns the nickname of the user being deowned.

Example

Message the channel where the event occurs: 
ON *:DEOWNER:#:msg # $nick $+ : You're not an owner of # anymore? :-o

Compatibility

Added: mIRC v5.61 (23 Sep 1999)
