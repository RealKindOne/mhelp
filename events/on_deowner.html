On Deowner



The ON DEOWNER event triggers whenever a user on a channel has been released of their owner (-q) status.

Synopsis

ON <level>:DEOWNER:<#[,#]>:<commands>

Parameters

Parameters


<level> - The corresponding {{mIRC|access levels}} for the event to trigger. * '''<*><#>''' - The place, or places where the event listens, you can specify specific name of window, seperate them by comma.
* - Any channel window ** '''#''' - Any channel window
<commands> - The commands to be performed when the event triggers == {{mIRC|local identifiers}} == === {{mIRC|$opnick}} === Returns the nickname of the user being deowned.

Example

Message the channel where the event occurs: 
ON *:DEOWNER:#:msg # $nick $+ : You're not an owner of # anymore? :-o
