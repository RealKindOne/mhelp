On Getfail



The ON GETFAIL event triggers whenever a remote user sends a file via {{mIRC|/dcc|DCC send}} and it fails.

Synopsis

ON <level>:GETFAIL::<commands>

Parameters


<level> - The corresponding {{mIRC|access levels}} for the event to trigger. * '''''' - The filename(s) that should match the received file. Seperate multiple values by commas. These can be {{mIRC|wildcard|wildcards}}. Example: ''*.txt,*.doc,my*.txt''
<commands> - The commands to be performed when the event listener's criteria is met. == {{mIRC|local identifiers}} == === {{mIRC|$filename}} === Returns the complete filename of the received file.

Example

Echo the name of the filename, along with both the user's nickname and address if available: 
ON *:GETFAIL:*:echo -a Failed to receive $filename $iif($nick,from $nick - ,from) Address: $address
