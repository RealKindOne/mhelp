On Help



The ON HELP event triggers whenever a user on a channel has been elevated to helper/halfop (mode +h) status in a channel.

Synopsis

ON <level>:HELP:<#[,#]>:<commands>

Parameters


<level> - The corresponding {{mIRC|access levels}} for the event to trigger. * '''<*><#>''' - The place, or places where the event listens, you can specify specific name of window, seperate them by comma.
* - Any channel window ** '''#''' - Any channel window
<commands> - The commands to be performed when the event triggers == {{mIRC|local identifiers}} == === {{mIRC|$hnick}} === Returns the nickname of the user being helped.

Example

Acknowledge when a user has been elevated to helper: 
ON *:HELP:#:msg # Congratulations, $hnick $+ ! You are now a helper thanks to $nick $+ .
