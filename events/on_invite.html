On Invite



The ON INVITE event triggers when a user sends the mIRC client and invite to a specific channel.

Synopsis

ON <level>:INVITE:<#[,#]>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<#[,#]> - The place, or places where the event listens, you can specify specific name of window, separate them by comma.
# - Any channel window.
#chan - Only in #chan window.
#chan,#chan2 - Only in #chan and #chan2 window.
<commands> - The commands to be performed when the event triggers.

Local Identifiers

$chan - The channel you are being invited into.
$nick - The invite sender.

Example

Echo to the active window the user and channel of the invite details: 
ON *:INVITE:#:echo -a $nick has extended an invite to $chan
Automatically join the channel from the invite: 
ON *:INVITE:#:join $chan

Compatibility

Added: mIRC v4.5 (30 Jun 1996)
