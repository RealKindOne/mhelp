On Mp3End



The ON MP3END event triggers whenever an [[MP3]] file has finished playing via the /splay command. This event fills the $filename identifier, which will be set to the complete location and filename of the sound file that just ended.

Synopsis

ON <level>:MP3END:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Echo to the active window when the sound file has ended, including its filename: 
ON *:MP3END:echo -a $nopath($filename) has ended
The example above will output results similar to the contents below: 
Chopin - Four Scherzos.mp3 has ended

Compatability

Added: mIRC v5.8 (05 Sep 2000)
