On Nosound



The ON NOSOUND event triggers when a remote user attempts to send a [[/sound command - mIRC|sound request]] and the file does not exist on the local mIRC system. This event fills the '''{{mIRC|$filename}}''' identifier with the file name of the attempted sound.

Synopsis

ON <level>:NOSOUND:<commands>

Parameters


<level> - The level for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Send a message to the user when sound file does not exist: 
ON *:NOSOUND:msg $nick Unfortunately, I do not have: $filename
