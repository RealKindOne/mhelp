On Op



The ON OP event triggers whenever a user on a channel has been elevated to a [[Channel operator - IRC|channel operator]]. This event also fills the {{mIRC|$opnick}} identifier with the nickname of the user who was added to [[Channel operator - IRC|Channel operator]] status. The $nick
identifier points to the user who added them.

Synopsis

ON <level>:OP:<#[,#]>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger..
<matchtext> - The text to match in the event..
<#[,#]> - Thechannel, or channels where the event listens. '''#'''Any channel if left by itself, otherwise refers to a specific channel
'''[,#]'''List of specific channel names

Example

Acknowledge when a user has been elevated to a channel operator: 
ON *:OP:#:msg # Congratulations, $opnick $+ ! $nick has made you a channel operator!
