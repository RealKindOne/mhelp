On Part



The ON PART event triggers when the mIRC client, or a remote user, parts/leaves a channel.

Synopsis

ON <level>:PART:<#[,#]>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<#[,#]> - The place, or places where the event listens, you can specify specific name of window, separate them by comma.
# - Any channel window
#chan - Only in #chan window
#chan,#chan2 - Only in #chan and #chan2 window
<commands> - The commands to be performed when the event triggers.

Example

Once the local mIRC client has left a channel, echo to the channel window the total users inside of the channel: 
ON ME:*:PART:#:echo # You have left # $+ .
The above example makes use of the On_Me event, which triggers only when the local mIRC client triggers the event, not remote users. When a user leaves, send them a notice: 
ON *:PART:#:msg # So sorry to see you go, $nick $+ !
