On ServClose



The ON SERVCLOSE event triggers within mIRC when a dcc fserve session ends.

Synopsis

ON <level>:SERVCLOSE:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<commands> - The commands to be performed when the event triggers
