On Signal



The ON SIGNAL event triggers after a call to the {{mirc|/signal}} command. This is a mechanism used to signal events in multiple script at the same time You can use {{mirc|$signal}} to refer to the name of signal triggering the event and you can use {{mIRC|$1-}} to refer to the parameters passed to /signal

Synopsis

ON <level>:signal:<matchtext>:<commands>

Parameters


<level> - The level for the event to trigger.
<matchtext> - The matchtext for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

ON *:signal:*:echo -s $signal -- $1-
