On Snotice



The ON SNOTICE event triggers when mIRC receives a server notice. This event fills the $1- identifier with the text from the server notice.

Synopsis

ON <level>:SNOTICE:<matchtext>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<matchtext> - The text that to be matched. Can also be a wildcard.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Halt the default server notice and echo it to the active window in plain text: 
ON ^*:SNOTICE:*: {
  echo -a * Server notice: $1-
  haltdef
}

Compatibility

Added: mIRC v3.8 (25 Nov 1995)
