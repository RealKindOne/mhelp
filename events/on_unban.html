On Unban



The on UNBAN event is triggered much like the On_Ban event, however it triggers when a ban is removed from a channel's ban list.

Synopsis

on <level>:UNBAN:<#[,#]>:<commands>

Parameters

<level> - The corresponding access levels for the event to trigger.
<#[,#]> - The place, or places where the event listens, you can specify specific name of window, separate them by comma.
# - Any channel window.
#chan - Only in #chan window.
#chan,#chan2 - Only in #chan and #chan2 window.
<commands> - The commands to be performed when the event triggers.

Example

; Unban event that listens on all channels
; and responds by messaging the channel.

on *:UNBAN:#:msg # Hey, look, the address $banmask was just removed from the channel's ban list!

Compatibility

Added: mIRC v4.72 (09 Jan 1997)
