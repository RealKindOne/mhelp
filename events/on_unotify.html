On Unotify



The ON UNOTIFY event triggers when a user in the [[/notify command - mIRC|notify list]] disconnects from the same server.

Synopsis

ON <level>:UNOTIFY:<commands>

Parameters


<level> - The level for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Halt mIRC's default notice and echo the event to the active window: 
ON ^*:UNOTIFY: {
  echo -a * $nick just left $network
  haltdef
}
