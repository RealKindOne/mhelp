On Usermode



The ON USERMODE event triggers when the local mIRC client's usermodes have been changed on a connection. This event fills the '''$1-''' identifier with the new usermodes for the client.

Synopsis

ON <level>:USERMODE:<commands>

Parameters


<level> - The level for the event to trigger.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Halt the default mIRC response, and then echo the new details to the active window: 
ON ^*:USERMODE: {
  echo -a * You have set new usermodes: $1-
  haltdef
}
