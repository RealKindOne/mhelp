On Wallops



The ON WALLOPS event triggers when an [[Irc operator - IRC|IRC Operator]] issues a network-wide important notice. This event fills the '''$1-''' and '''{{mIRC|$nick}}''' identifiers with the full text of the [[Wallop - IRC|WALLOP]], and the operator's name who issued it, respectively.

Synopsis

ON <level>:WALLOPS:<matchtext>:<commands>

Parameters


<level> - The level for the event to trigger. '''<matchtext>'''The text that to be matched. Can also be a {{mirc|wildcard}}.
<commands> - The commands to be performed when the event listener's criteria is met.

Example

Echo to the active window the current date/time, the IRC operator's name who sent the WALLOP, the WALLOP message, and prevent the default response: 
ON ^*:WALLOPS:*: {
  echo -a [[ $asctime ]] ==> WALLOP From $nick :: Message:
  haltdef
}
