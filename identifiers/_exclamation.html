$!



The $! identifier returns the input value in an input request ($input, $? etc). This identifier return the same value as the identifier used. Only beneficial to avoid saving the result of $input etc to a variable.

Synopsis

$!

Parameters

None

Properties

None

Example

After you execute the alias, via /testme, you will be prompted by an input box. Whatever input you put in that box, upon clicking OK, will be immediately echoed to your active window. 
alias testme {
  echo -ag $?="Type something" is equal to $!
  echo -ag $input(click no,yvn) is equal to $!
}
