$+



$+ is a construct, it has the look of an identifier but does not return a value. It concatenates two things together. In addition to its normal behavior, $+ has a few special functions when used within evaluation brackets [ ] $+() on the other hand, is an identifier, taking parameters, where the returned value is just the concatenation of the passed parameters.

Synopsis

value $+ value1 $+ etc $+(value,value1,etc)

Parameters

You can pass any number of parameter.

Examples

echo -a A $+ B $+ C
Will result in: ABC 
var %x = A, %y = B
echo -a %x $+ %y $+ $day
Will print (depending on the day of the week) "ABMonday" -- Same example with $+(): 
var %x = A, %y = B
echo -a $+(%x, %y, $day)
