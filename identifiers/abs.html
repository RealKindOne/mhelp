$abs



The $abs identifier returns the absolute value of a number.

$abs support scientific notation input, you can use the letter 'e' or 'd' as *10^N, $abs(5e6) = $abs($calc(5*10^6))

Synopsis

$abs(N)

Parameters

N - The number that you want to retrieve the absolute value for.

Properties

None

Example

Echo the absolute value of -7 to the active window: 
//echo -a $abs(-7)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
