$active



The $active identifier will return the full name of the currently active window in mIRC, excluding all dialogs and the script editor. Some window name can have spaces in them, namely "Status Window" can be returned, if you use $active in your script inside command taking a window parameter, don't forget to use $qt($active).

Synopsis

$active

Paramaters

None

Properties

None

Example

Echo the value of $active to the active window 
//echo -a $active

Compatibility

Added: mIRC v4.5 (30 Jun 1996)
