$activecid

$activecid



The $activecid identifier returns the connection ID related to the active window. This means that if the window was created on connection 39, the $activecid for that window would be "39".

Synopsis

$activecid

Parameters

None

Properties

None

Example

Echo to the currently active window the connection ID it is related to 
//echo -a $activecid

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

Notes

Connection ID and the Nth connections are different value, your first status window can have ID 39 instead of ID 1.
