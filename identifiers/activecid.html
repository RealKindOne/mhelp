$activecid



The $activecid identifier returns the connection id related to the active window. This means that if the window was created on connection 1, the $activecid for that window would be "1".

Synopsis

$activecid

Parameters

None

Properties

None

Example

Echo to the currently active window the connection ID it is related to 
//echo -a $activecid

Compatibility

Added: mIRC v6.0 (03 Feb 2002)
