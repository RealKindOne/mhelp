$activewid



The $activewid identifier returns the window ID for the currently active window. Windows ID uniquely represent a window and can be used in a window name as @IDto represent that window instead of using the window name.

Synopsis

$activewid

Parameters

None

Properties

None

Example

Echo the current window's id

//window @test | aline @ $+ $activewid test
