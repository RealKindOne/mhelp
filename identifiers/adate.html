$adate

$adate



The $adate identifier returns the current date in American month/day/year format compared to $date returning day/month/year.

Synopsis

$adate

Parameters

None

Properties

None

Example

Echo the current date to the active window in both $adate and $date: 
//echo -a Today in mm/dd/yyyy format is $adate - today in dd/mm/yyyy format is $date

Compatibility

Added: mIRC v3.9 (28 Dec 1995)

None
