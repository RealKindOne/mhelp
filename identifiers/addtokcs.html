$addtokcs



The $addtokcs identifier adds a token to a list of token, only if the token does not already exist in the list case sensitively, return the resulting list. $addtok is case insensitive version.

Synopsis

$addtokcs(<list>,<token>,<C>)

Parameters

<list> - The list of tokens (can be empty).
<token> - The token to add, if you pass $null here, the original list is returned, it doesn't return an error.
<C> - The ascii number (or code point) of the character separating the tokens.

Properties

None

Example

Display "A a", $addtok would have returned "A". 
//echo -a $addtokcs(A,a,32)
