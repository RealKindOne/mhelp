$agentname

$agentname



The $agentname can be used in conjunction with the On_agent in order to return the name of the Agent associated with the event.

Synopsis

$agentname

Parameters

None

Properties

None

Example

After the on AGENT Event triggers, return the name of the Agent that just finished speaking: 
on *:AGENT:echo -a $agentname has just finished speaking.

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
