$anick

$anick



The $anick identifier returns the value of your alternative nickname, which is set in the mIRC options dialog or using the /anick command.

Synopsis

$anick

Parameters

None

Properties

None

Example

Echo the value of $anick to the active window 
//echo -a $anick

Compatibility

Added: mIRC v5.8 (05 Sep 2000)

Notes

None

See also

$me
$mnick
/anick
/mnick
