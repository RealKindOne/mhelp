$ansi2mirc



The $ansi2mirc identifier converts ANSI colors into mIRC colors.

Synopsis

$ansi2mirc(<ANSI>)

Parameters

<ANSI> - input text containing eventual ANSI color to be converted

Properties

None

Example

Echo some ANSI code, converted, to the active window 
//echo -ga $ansi2mirc(this is probably $chr(27) $+ [32mgreen $+ $chr(27) $+ [0m.)

Compatibility

Added: mIRC v5.5 (08 Jan 1999)
