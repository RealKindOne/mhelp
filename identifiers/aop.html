$aop



The $aop identifier returns the status of auto-op, $true for enabled, $false for disabled.

Synopsis

$aop

Parameters

None

Properties

None

Example

Echo to the active window whether or not auto-op is enabled: 
//echo -a $aop

Added: mIRC v5.9 (26 Apr 2001)
