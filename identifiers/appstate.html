$appstate



The $appstate''' identifier returns the current state of the mIRC application window.

Synopsis

$appstate

Properties

None == Results == '''full'''The mIRC application window size is full. '''hidden'''The mIRC application window size is hidden. '''maximized'''The mIRC application window size is maximized. '''minimized'''The mIRC application window size is minimized. '''tray'''The mIRC application window has been minimized to the Windows tray.

Example

'''Echo the mIRC application window state to the active window''' 
//echo -a $appstate

Compatibility

Added: mIRC v5.7 (02 Feb 2000)
