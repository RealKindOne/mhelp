$argon2



The $argon2 identifier calculates the Argon2 key derivation function for password hashing. Argon2 is a modern, memory-hard KDF designed to resist GPU and ASIC attacks.

Synopsis

$argon2(text|&binary, salt|&binary, hash, length, iterations, parallelism, memcost, [opts])

Parameters

text|&binary - The password input as text string or binary variable
salt|&binary - The salt value as text string or binary variable
hash - Argon2 variant: 2i (data-independent), 2d (data-dependent), or 2id (hybrid)
length - Desired output key length in bytes 1 - 2048
iterations - Time cost (number of iterations) 1 - 9
parallelism - Degree of parallelism (number of threads) 1 - 9
memcost - Memory cost in KiB (kilobytes) 0 - 512
opts - Optional: b to indicate both password and salt are binary variables

Properties

None

Example

Echo the Argon2id hash of a password with salt: 
//echo -a $argon2(myPassword, mySalt, 2id, 32, 3, 4, 512)
