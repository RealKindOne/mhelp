$asc



The $asc identifier returns the code point of the character (or first character of the string).

Synopsis

$asc(<char|string>)

Parameters

char - The character you want the code point of, up to 65535. If you want to learn more about why 65535 is the limit and/or if you want to get the codepoint of an unicode character whose codepoint is greater than that, see the Unicode page

Example

Echo the ascii value/code point for the letters ''A'' and ''a'' to the active window
//echo -a A is $asc(A) and a is $asc(a)
Echo the Unicode codepoint for the lower-case 'm' because that's the first character of the string.
//echo -a $asc(mIRC)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)
