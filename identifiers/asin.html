$asin



The $asin identifier return the arcsine of N.

Synopsis

$asin(N)[.deg]

Parameters

N - The number you want the arc sine of, in radian.

Properties


.deg - means the N parameter value is in degree

Example

//echo -a $asin(5)

Compatibility

Added: mIRC v5.3 (13 Dec 1997)

Notes

None
