$auto()

$auto()



The $auto() identifier allows you to get data from your /auto (Auto-Op) list in mIRC.

Note: This feature is depreciated and has been replaced with $aop().

Synopsis

$auto(address|N)[.property]

Parameters

address - Returns any match based on the specified address from the auto-op list.
N - Returns any match in the auto-op list based on the Nth entry in the list. If N is 0, the total amount of addresses in the auto-op list will be returned.

Properties

.type - If a match is found, returns the list of channels associated with its auto-op.
.network - Returns which network the auto-op is associated with.

Example

Echo to the active window the total number of addresses in the auto-op list: 
//echo -a $auto(0)
Echo to the active window the network that the match is associated with: 
//echo -a $auto(*testhost.com).network

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
