$avoice

$avoice



The $avoice identifier returns the status of auto-voice, $true for enabled, $false for disabled.

Synopsis

$avoice

Parameters

None

Properties

None

Example

Echo to the active window whether or not auto-voice is enabled: 
//echo -a $avoice

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
