$awaymsg



The $awaymsg identifier will return the message you set up when you originally used the /away command.

Synopsis

$awaymsg

Parameters

None

Properties

None

Example

Echo your away msg to the active window 
//echo -a $awaymsg

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

Notes

None
