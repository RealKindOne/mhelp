$awaytime



The $awaytime identifier will return how long you've been away since using the /away command.

Synopsis

$awaytime

Parameters

None

Properties

None

Example

Echo the amount of time you have been away to the active window. 
//echo -a $awaytime

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

Notes

None
