$beta

$beta



The $beta identifier returns the beta portion of the version number if you're running a beta. Otherwise returns $null string.

Synopsis

$beta

Parameters

None

Properties

None

Example

//echo -a version is $version $+ $iif($beta,. $+ $beta)
When someone on forum is having trouble with their mIRC acting incorrectly, it's often part of the info string they're requested to paste in reply: 
//echo -a $os $version $beta $md5($mircexe,2) $file($mircexe).sig $alias(0) $script(0) $dll(0) $com(0)
Prevent a script from running on a beta version: 
on *:start:{
  if ($beta) {
    echo -as WARNING: Beta version of mIRC detected. Not running script.
    return
  }
  ; Rest of script code here.
}

Compatibility

Added: mIRC v5.7 (02 Feb 2000)

Notes

Version before 7.43 would return "betaN" where "N" is a number. All version after only return a integer.

Version before 6.35 would return $false instead of $null if you were not running a beta.
