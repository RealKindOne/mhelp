$bindip



The $bindip identifier is used to get information regarding an available [[network adapter]].

Synopsis

$bindip(N|ipaddress)[.property]

Parameters


N - Returns the Nth device in the network adapters list. If ''N'' is specified as ''0'', returns the total number of active network adapters.
ipaddress - If you specify an IP address, mIRC will try to find the best network adapter to use to connect to that IP.

Properties


.name - The name of the network adapter.
.ip - The IP address of the network adapter.
.loopback - Returns ''$true'' if the IP/device has a [[loopback]], otherwise ''$false''. == Examples == '''Echo to the active window the total number of network adapters''' 
//echo -a $bindip(0)
'''Echo to the active window the name of a local network adapter''' 
//echo -a $bindip(192.168.1.1).name

Compatibility

Added: mIRC v7.0 (02 Apr 2010)
