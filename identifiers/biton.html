$biton



The $biton identifier returns N with the Bth bit set to 1.

Synopsis

$biton(<N>,<B>)

Parameters


N - the number, in base10/decimal, if N is 2^32 or greater, 2^32-1 will be used as the number. If N -2^32 or less, 2^32-1 is also used as the number
B - The Bth bit to turn on

Properties

None

Example

//var %n 10 | echo -a $biton(%n,3)
; returns 14

Here's how the answer was calculated:
# Convert base-10 N to binary.
# Count 3 from the right
# that bit is set to 1.
# Return the possibly altered number //var %n 10 | echo -a $base(%n,10,2,32) | echo -a $str(-,24) $+ 87654321 | echo -a $base($biton(%n,3),10,2,32) $base(1110,2,10) ; returns: 00000000000000000000000000001010 ------------------------87654321 00000000000000000000000000001110 14 Using $biton and $bitoff, you can store several variables as bits within a single variable, instead of creating a separate variable for each.
The /window switch -wN uses bit settings, with default 3 if N not used. 
var %treebarsetting $true
var %switchbarsetting $true
var %n
if (%switchbarsetting) var %n $biton(%n,1)
if (%treebarsetting) var %n $biton(%n,2)
if (%n == $null) var %n 3

; $biton can be simulated by using $or to set the appropriate bit to 1
//var %n 10 | var %b 3 | echo -a within the valid number range, %n $biton(%n,%b) is the same as $or( %n , $calc( 2^ (%b -1) ) )
