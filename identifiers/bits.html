$bits



The $bits identifier returns the bit version for the mIRC application you are using; either 32 or 16.

Synopsis

$bits

Parameters

None

Properties

None

Example

Echo mIRC's application bit number to the active window 
//echo -a $bits

Compatibility

Added: mIRC v5.31 (04 Jan 1998) Added on: 04/01/98

Notes

None
