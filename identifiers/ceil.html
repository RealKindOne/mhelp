$ceil



The $ceil''' identifier will calculate and return a numerical value rounded to the next highest integer.

Synopsis

$ceil(N)

Parameters


N - The number to be rounded up. Decimals cause the value to round up, while integers are returned the same. You can use the letter 'e' or 'd' as *10^N, $ceil(5e6) = $ceil($calc(5*10^6))

Example

Echo to the active window '''3.14''' rounded to the next highest integer: 
//echo -a $ceil(3.14)

Compatibility

Added: mIRC v6.1 (29 Aug 2003)
