$chan

$chan



The $chan identifier is filled during events which trigger on a channel or is filled if the active window is a channel.

Synopsis

$chan

Parameters

None

Properties

None

Example

Echo into your current active window that someone said something in a channel. 
on *:TEXT:*:#:{
  echo -a $nick said something in $chan
}
