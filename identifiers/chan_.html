$chan()

$chan()



The$chan() can be used to get information regarding a specific channel that is open.

Synopsis

$chan(@wid/#/N)[.property]

Parameters

(@wid/#/N) - The window ID of the channel, a specific channel, or a channel number referencing the channel.

Properties

.banlist - Returns $true or $false if the internal ban list has been updated for the channel or $inmode if the list is being updated
.cid - Returns the connection ID for the channel.
.hwnd - Returns the window handle for the channel.
.ial - Returns $true or $false if the internal address list has been updated for the channel or returns $inwho if still processing /who output.
.ibl - Returns the same as .banlist
.iql - Returns $true or $false if the internal quiet list has been updated for the channel or $inmode if the list is being updated
.idle - Returns the number of seconds since you joined the channel or another person said something, whichever happened last.
.iel - Returns $true or $false if the internal exempt list has been updated for the channel or $inmode if the list is being updated
.iil - Returns $true or $false if the internal invite list has been updated for the channel or $inmode if the list is being updated
.inwho - Returns $true or $false if a /who has been performed on the channel, and the results are still being processed.
.key - Returns the channel's key.
.limit - Returns the amount of users the channel is limited to.
.logfile - Returns the full logfile name if a log is opened for the window, otherwise returns $null.
.mode - Returns the current modes set in the channel.
.stamp - Returns either $true, or $false depending on if time-stamping is enabled or disabled.
.status - Returns: joining, joined, kicked, or parted.
.topic - Returns the channel topic.
.wid - Returns the window ID for the channel window.

Example

Create an alias that opens a custom window which loads some basic information about the active channel: 
alias getDetails {
  if ($active ischan) {
  var %c = $active
    window @getDetails 350 350 650 200
    clear @getDetails
    echo @getDetails Channel: %c
    echo @getDetails $crlf $crlf
    echo @getDetails Topic: $chan(%c).topic
    echo @getDetails $crlf $crlf
    echo @getDetails Modes: $chan(%c).mode
    echo @getDetails $crlf $crlf
    echo @getDetails Key: $iif($chan(%c).key,$v1,No key set)
  }
}
This alias can now be executed using the following command in a channel command-line: 
/getDetails
