$chanmodes



The $chanmodes''' identifier will return the supported list of channel modes available on the active network connection. If not connected, mIRC defaults to a standard value of ''bIe,k,l''. '''Note:''' Any active connection window can use this identifier, not just an active channel.

Parameters

None

Example

Echo the current available channel modes to the active window: 
//echo -a $chanmodes

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
