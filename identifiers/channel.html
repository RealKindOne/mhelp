$channel



The $channel identifier returns informations about channels.

Note: This is depreciated. use $chan.

Synopsis

$channel(N/#)

Parameters


N/# - The channel or the Nth channel, if N is 0, returns the total numbers of channels

Properties

See $chan for more informations.

Example

//echo -a $channel(3).mode

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
