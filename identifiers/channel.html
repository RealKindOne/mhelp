$channel



The $channel identifier returns information about channels.

Note: This feature is depreciated and has been replaced with $chan.

Synopsis

$channel(N/#)[.property]

Parameters

N/# - The channel or the Nth channel, if N is 0, returns the total numbers of channels

Properties

See $chan for more information.

Example

//echo -a $channel(3).mode

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
