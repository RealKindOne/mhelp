$chr



The $chr identifier returns the character associated with a code point

Synopsis

$chr(<N>)

Parameters


N - The code point, up to 65535 If you want to learn more about why 65535 is the limit and/or if you want to get an unicode character whose codepoint is greater than that, see the Unicode page

Properties

None

Example

Echo the character attached to ascii "63" to the active window 
//echo -a $chr(63)
Echo the character attached to ascii "85" to the active window 
//echo -a $chr(85)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
