$chr



The $chr identifier returns the character attached to a specified ascii number.

Synopsis

$chr(<N>)

Parameters


N - This is the value that you wish to retrieve the character for.

Properties

None

Example

Echo the character attached to ascii "63" to the active window 
//echo -a $chr(63)
Echo the character attached to ascii "85" to the active window 
//echo -a $chr(85)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
