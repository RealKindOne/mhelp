$cid



The $cid identifier returns the current connection ID. Cid uniquely represent a status window/connection.

Synopsis

$cid

Parameters

None

Properties

None

Example

Echo to the currently active window the connection ID it is related to 
//echo -a $cid

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

Notes

None
