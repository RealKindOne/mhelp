$clevel



The $clevel identifier will return the matching event level for any triggered [[List of on events - mIRC|ON Events]].

Synopsis

$clevel

Parameters

None

Properties

None

Example

When a user speaks in a channel, echo the event level that is triggered: 
ON *:TEXT:*:#:echo -a Event level triggered: $clevel

Compatibility

Added: mIRC v4.6 (07 Sep 1996)

Notes

None
