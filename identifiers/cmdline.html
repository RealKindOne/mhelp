$cmdline



The $cmdline return the value passed to the executable when mIRC is ran.

Synopsis

$cmdline

Parameters

None

Properties

None

Example

on *:start:{
  if ($cmdline == debug) { echo -a debug }
}
;//run $mircexe debug

Compatibility

Added: mIRC v6.0 (03 Feb 2002)
