$codepage

$codepage



The $codepage identifier allows you to list all the available codepages and/or get information about a given codepage. While mIRC went away from codepage support for unicode, it was merely to go away from all the troubles mIRC had trying to resolve multiple encoding at the same time. Unicode is just another encoding and with mIRC being clean for unicode meant adding codepage back was much easier, mIRC support codepage back and the codepage ID can be used directly with /server to specify an encoding for that connection (-encoding directive)

Synopsis

$codepage(N/ID)[.property]

Parameters

N/ID - The Nth codepage to be returned, or 0 for the total number of codepage in the list, you can also pass an ID.
Note: first ID starts at 437 and only increases from that, while there's a total of only 58 codepages currently, and there should be no overlap given no new codepage are added.

Properties

.name - Returns the name of the codepage
.desc - Returns the description of the codepage.
.id - Returns the ID of the codepage (default).

Example

//var %a 1 | while ($codepage(%a)) { echo -sg $v1 -- $codepage(%a).name -- $codepage(%a).desc -- $codepage(%a).id  | inc %a }
