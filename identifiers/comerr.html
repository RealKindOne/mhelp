$comerr



The $comerr identifier return 1 if an error occured with a {{mIRC|COM|COM object}}, 0 otherwise. This identifier should be checked after any $com / $comcall call.

Synopsis

$comerr

Paramters

None

Properties

None

Example

//comopen a nonexistant | echo -a $comerr

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
