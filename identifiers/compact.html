$compact

$compact



The $compact identifier is used to find out if mIRC is or is not in compact state.
If the response is $true then mIRC is in compact mode.

Synopsis

$compact

Parameters

None

Properties

None

Example

Echo whether or not mIRC is in compact mode to the active: 
//echo -a $compact

Compatibility

Added: mIRC v6.32 (21 May 2008)
