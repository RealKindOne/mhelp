$creq



The $creq identifier returns the current parameter settings for DCC Chat requests.

Synopsis

$creq
== Switches == None == Results == '''ask'''DCC Chat requests will popup an authorization window asking you whether you wish to accept or decline all incoming DCC Sends. '''auto'''All incoming DCC Chats will be automatically accepted. '''ignore'''All incoming DCC Chats will be ignored.

Example

Echo the current status of DCC Chats to the active window: 
//echo -a $creq

Compatibility

Added: mIRC v5.4 (23 Jun 1998)}

Notes

None
