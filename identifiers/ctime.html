$ctime



The $ctime allows you to retrieve the total number of seconds elapsed since 00:00:00 GMT, January 1, 1970 based on your system time.

Synopsis

$ctime

Parameters

None

Properties

None

Example

Echo the current $ctime to the active window: 
//echo -a $ctime

Compatibility

Added: mIRC v4.5 (30 Jun 1996)
