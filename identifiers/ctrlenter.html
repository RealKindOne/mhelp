$ctrlenter

$ctrlenter



The $ctrlenter identifier returns either $true or $false depending on whether or not an On_Input event was triggered using the "CTRL+ENTER" combination.

Synopsis

$ctrlenter

Parameters

None

Properties

None

Example

Echo to whether or not "CTRL+ENTER" was used in an input box: 
on *:INPUT:*: { echo -a $ctrlenter }
Make a script do nothing if CTRL+Enter is used. 
on *:INPUT:*:{
  ; Do nothing if CTRL+ENTER is pressed.
  if ($ctrlenter) { return }
  ; Do stuff if CTRL+ENTER is not pressed.
  echo -a CTRL+ENTER not pressed.
}
