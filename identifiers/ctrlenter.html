$ctrlenter



The $ctrlenter''' identifier returns either $true
or $false
depending on whether or not an [[On input - mIRC|ON INPUT]] event was triggered using the '''CTRL+ENTER''' combination.

Synopsis

$ctrlenter

Parameters

None

Example

Echo to whether or not '''CTRL+ENTER''' was used in an input box: 
ON *:INPUT:*: { echo -a $ctrlenter }

Compatibility

Added: mIRC v5.7 (02 Feb 2000)
