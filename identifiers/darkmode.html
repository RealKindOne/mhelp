$darkmode

$darkmode



The $darkmode identifier returns $true if mIRC is currently following the Windows dark mode state.

Synopsis

$darkmode

Parameters

None

Properties

None

Example

Echo Dark Mode state: 
//echo -a $darkmode

Compatibility

Added: mIRC v7.79 (03 Feb 2025)

Notes

Dark Mode only works on Windows 10 and 11.
