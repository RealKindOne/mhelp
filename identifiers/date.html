$date

$date



The $date identifier can be used to return the current date in day/month/year format.

Synopsis

$date

Parameters

None

Properties

None

Example

Echo today's full date to the active window in dd/mm/yyyy format: 
//echo -a $date

Compatibility

Added: mIRC v3.5 (07 Aug 1995)

None
