$daylight



The $daylight identifier will return the seconds offset for daylight savings. If daylight savings is not in effect, returns "0".

Synopsis

$daylight

Parameters

None

Properties

None

Example

Echo the daylight savings offset to the active window: 
//echo -a $daylight

Compatibility

Added: mIRC v5.4 (23 Jun 1998)

Notes

None
