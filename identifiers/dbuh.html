$dbuh



The $dbuh identifier returns what the [[DBU mIRC|DBU]] per pixel height value is for the current display.

Synopsis

$dbuh

Parameters

None

Properties

None

Example

Echo the value of '''$dbuh''' to the active window: 
//echo -a $dbuh
