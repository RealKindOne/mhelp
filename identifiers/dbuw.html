$dbuw



The $dbuw identifier returns what the [[DBU mIRC|DBU]] per pixel width value is for the current display.

Synopsis

$dbuw

Parameters

None

Properties

None

Example

Echo the value of '''$dbuw''' to the active window: 
//echo -a $dbuw

Compatibility

Added: mIRC v5.7 (02 Feb 2000)
