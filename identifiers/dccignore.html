$dccignore

$dccignore



The $dccignore identifier will return either $true or $false depending on the the /dcc ignore option within mIRC.

Synopsis

$dccignore

Parameters

None

Properties

None

Example

Echo $true if $dccignore is enabled: 
//echo -a $dccignore

Compatibility

Added: mIRC v5.61 (23 Sep 1999)

Notes

None

See also

$dccignore()
/dcc
