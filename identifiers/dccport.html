$dccport



The $dccport identifier returns the current port number being used by the /dcc server command.

Synopsis

$dccport

Parameters

None

Properties

None

Example

Echo to the active screen the port being used by the /dcc server command. 
//echo -a $dccport

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
