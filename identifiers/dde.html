$dde



The $dde identifier returns the value returned by the specified service name, topic, and item, by sending an XTYP_REQUEST.

Synopsis

$dde(name, topic, item, delay)

Parameters

name - The service name
topic - The topic name
item - The item name (optional)
delay - if specified mIRC will wait <delay> seconds for a reply before giving up, default is 1 second (optional)

Properties

None '''Note''': If the value returned by $dde() is too long for mIRC to handle, $dde returns a value of plain text "$error". '''Note2''': $dde supports one of the old syntax for some identifiers, evaluating '$dde name topic item' works (use 0 if no item, seems to be required sometimes), there's even a -dN switch for the delay: should return the $server value you are connected to.

Example

//echo My other mIRC is $dde(mirc, connected) to $dde(mirc, server)

Compatibility

Added: mIRC v3.9 (28 Dec 1995)

Notes

None
