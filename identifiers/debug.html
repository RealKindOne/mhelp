$debug

$debug



The $debug identifier will return either the window name or the filename of the current connection's debug output target.

Synopsis

$debug

Parameters

None

Properties

None

Example

Echo to the active screen the current, if any, debug target: 
//echo -a $iif($debug,$v1,None)

Compatibility

Added: mIRC v6.0 (03 Feb 2002)
