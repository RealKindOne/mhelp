$deltok

$deltok



The $deltok identifier deletes the Nth token from text.

Synopsis

$deltok(list,N-N2,C)

Parameters

<list> - The list of tokens (can be empty)
<N-N2> - The Nth tokens to delete
<C> - The ascii number (or code point) of the character seperating the tokens If you pass $null for N-N2, the original list is returned. If you pass a single number N, it represents a single token to delete, passing 0 will return the original string. Otherwise, it's a range of tokens to delete in the format N-N2: -Both N and N2 can be negative, in this case the position they refer to starts from the end of the string instead. $deltok(a b c d e f g,-3--5,32) will delete the tokens from position -5 (c) to position -3 (e) resulting in "a b f g" -Both N and N2 can be 0, in which case they refer to the first position, same as using 1. -N2 can be smaller than N, $deltok will work it out: $deltok(string,2-1,C) = $deltok(string,1-2,C) -N2 is optional, $deltok(string,N-,C) will delete all tokens onward, starting at the Nth position. -The character '-' used to separate the two numbers can be spaced out N - N2, no need to use $+ to build dynamic ranges.

Properties

None

Example

//echo -a $deltok(a b,1,32)
Would display "b"
