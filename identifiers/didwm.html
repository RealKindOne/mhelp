$didwm



The $didwm''' identifier return the number of the line that matched the wildcard expression in dialog's control.

Synopsis

 $didwm(name,id,wildcard,N)

Parameters


name -name of the dialog * '''id''' - id of the control, can be a list, editbox, and combo
N - optional, if N is passed, start matching at the Nth line * '''wildcard''' - the wildcard expression.

Properties

Example

None
