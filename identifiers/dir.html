$dir



The $dir identifier displays the select folder dialog and returns the selected folder.

Note: This feature is depreciated and has been replaced with $sdir.

Synopsis

$dir="title" <dir>

Parameters

title - The optional title of the dialog, the quote are optional if your title does not have space
<dir> - the directory/file to be opened, can be a wildcard

Properties

None

Example

//echo -a $dir='select dir' c:\
