$dllcall



The $dllcall''' call a function of a dll, just like {{mIRC|$dll}}, but is 'multithreaded', it won't halt the script and will call the specified alias once the call returns. $dllcall does not return a value.

Synopsis

$dllcall(name.dll,alias,procname,data)

Parameters

* '''name.dll''' - the filename to the dll * '''alias''' - The alias to be called once the function call returns, the alias is called with one parameter: the dll complete filename * '''procname''' - procname, the name of the function in the dll * '''data - a string/parameter passed to the function
