$dname

$dname



The $dname identifier is used within an On_Dialog event to get the name of the dialog that triggered the event.

Synopsis

$dname

Parameters

None

Properties

None

Example

When any dialog opens, echo the name of the dialog to the active window: 
ON *:DIALOG:*:init:0: echo -a $dname

Compatibility

Added: mIRC v5.5 (08 Jan 1999)

None
