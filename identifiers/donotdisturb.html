$donotdisturb



The $donotdisturb identifier returns $true if enabled or $false if disabled.

Synopsis

$donotdisturb

Parameters

None

Properties

None

Example

//echo $donotdisturb

Compatibility

Added: mIRC v7.62 (29 Jul 2020)

Notes

This will always return $false on Windows XP as the feature does not work on that OS.
