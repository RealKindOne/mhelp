$email

$email



The $email identifier returns the email address you have specified in the mIRC connect dialog.

Note: This feature is depreciated and has been replaced with $emailaddr.

Synopsis

$email

Parameters

None

Properties

None Echo the email address you provided to the currently active window:

Example

//echo -a $email

Compatibility

Added: mIRC v5.1 (28 Aug 1997)

Notes

None
