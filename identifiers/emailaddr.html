$emailaddr



The $emailaddr identifier returns the email address you have specified in the mIRC connect dialog.

Synopsis

$emailaddr

Parameters

None

Properties

None

Example

Echo the email address you provided to the currently active window: 
//echo -a $emailaddr

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

Notes

None
