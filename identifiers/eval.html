$eval



The $eval identifier allows you to dynamically evaluate a line of code N times.

Synopsis

$eval(text,N)

Parameters

text - The text you want to double evaluate.
N - The number of time you want to evaluate something, if N = 0, the text is not evaluated,

Properties

None

Example

//var %prefix_ $+ $me value | echo -a $eval($+(%,prefix_,$me),2)

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

Notes
