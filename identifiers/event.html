$event

$event



The $event identifier will return the event name of any triggered On_Events. It will also be filled inside raw events with the numeric or string event's name

Synopsis

$event

Parameters

None

Example

When a user speaks in a channel, echo the event name that is triggered: 
ON *:TEXT:*:#:echo -a Event triggered: $event

Compatibility

Added: mIRC v5.3 (13 Dec 1997)

Notes

None
