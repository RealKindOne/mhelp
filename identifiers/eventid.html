$eventid



The $eventid is a local identifier which is filled with a unique ID inside any event in mIRC. This identifier allows scripters to use SendMessage() in an event context, in order to evaluate local identifiers (such as $nick inside an On_Text event). Theoretically, this unique ID can be used for anything.
