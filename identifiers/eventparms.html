$eventparms



The $eventparms identifier returns the event-specific parameters passed by the script parser to an executed command. This is nothing but a version of $parms that is 'local', it only exists during an event and it has a global scope: you can access the value from an alias called from the event.

Synopsis

$eventparms

Parameters

None

Example

on *:text:*:#:echo -s $parms -- $eventparms | testpm
alias testpm echo -s $parms -- $eventparms
