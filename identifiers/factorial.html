$factorial



The $factorial identifier is a math identifier which returns the factorial of the integer number N In bigfloat mode, the maximum number for which you can get the factorial of is 3336 In normal mode, the maximum number is 170

Synopsis

$factorial(<integer>)

Parameters


<integer<
The integer you want the factorial of

Example

//echo -ag $factorial(70)

Compatibility

Added: mIRC v7.72 (27 Nov 2022)

Notes

None
