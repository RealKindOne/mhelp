$false

$false



The $false is returned during any statement that is either written for it to return a value of $false, or performing a comparison operation where the outcome is $false.

Synopsis

$false

Parameters

None

Example

Echo to the active window a result that returns $false 
//echo -a $isfile($mircexee)

Compatibility

Added: mIRC v2.1a (28 Feb 1995)
