$ferr



The $ferr''' return 1 if a file error of some kind occured from the last file access attempt in any script, 0 otherwise

Synopsis

$ferr

Parameters

None

Properties

None

Example

//fopen test noexist | echo -a $ferr | fclose test

Added: mIRC v6.1 (29 Aug 2003)
