$fgetc



The $fgetc identifier returns the next character from an open file in file handling.

Synopsis

$fgetc(name/N)

Parameters


N - The Nth handle to be used
name -The name of the handle to be used

Properties

None

Example

//fopen test $qt($mircini) | echo -a > $fgetc(test) | fclose test

Compatibility

Added: mIRC v6.1 (29 Aug 2003)

Notes

None
