$finddirn



The $finddirn identifier returns the Nth position of the found directory inside a $finddir call executing commands.

Synopsis

$finddirn

Parameters

Nonen

Properties

None

Example

//echo -a $finddir($mircdir,*,0,0,echo -a $finddirn : $1-)

Compatibility

Added: mIRC v6.17 (17 Feb 2006)

Notes

None
