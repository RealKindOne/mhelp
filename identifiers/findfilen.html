$findfilen



The $findfilen identifier returns the Nth position of the found file inside a $findfile call executing commands.

Synopsis

$findfilen

Parameters

Nonen

Properties

None

Example

//echo -a $findfile($mircdir,*,0,0,echo -a $findfilen : $1-)

Compatibility

Added: mIRC v6.17 (17 Feb 2006)

Notes

None

$finddirn
