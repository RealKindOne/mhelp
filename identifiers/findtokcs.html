$findtokcs



The $findtokcs identifier returns the position of the Nth matching token in text using case sensitivity. $findtok is a case insensitive equivalent.

Synopsis

$findtokcs(<list>,token,N,C)

Parameters

<list> - The list of tokens (can be empty).
token - The token used to match.
N - The Nth matching token, default to 1, if N is 0, returns the total number of matching tokens. Cannot be negative, but this parameter can be omitted entirely.
<C> - The ascii number (or code point) of the character separating the tokens.

Properties

None

Example

Return 4 
$findtokcs(one two three twO, twO, 32)

Compatibility

Added: mIRC v5.91 (15 Jun 2001)
