$freadex



The $freadex identifier returns the total content of the file from the pointer.

Synopsis

$freadex(N/name) returns the content of the file

Parameters

N - The Nth handle to be used
name -The handle name to be used

Properties

None

Example

//fopen test $qt($mircini) | echo -a > $freadex(test) | fclose test

Compatibility

Added: mIRC v7.59 (08 Feb 2020)

Notes

None
