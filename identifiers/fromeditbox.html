$fromeditbox



The $fromeditbox identifier returns $true or $false depending on if the alias was called from a mIRC editbox.

Synopsis

$fromeditbox

Parameters

None

Properties

None

Example

define the alias testedit: 
alias testedit echo -a $fromeditbox
and then: 
//echo -a $fromeditbox | testedit

Compatibility

Added: mIRC v7.52 (28 Feb 2018)

Notes

None
