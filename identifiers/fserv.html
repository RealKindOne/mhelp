$fserv

$fserv



The $fserv identifier can be used to gather various details regarding open /fserve window.

Note: This feature is depreciated and has been replaced with $fserve.

Synopsis

$fserv(N/nick[,N])[.property]

Parameters

N/nick - Refers to either the Nth open connection, or the connection associated with a specific user's nickname.
[N] - If a nickname is specified in the primary parameter, the secondary N targets the Nth connection associated with that nickname match.

Properties

.cd - The current directory of the Fserve.
.cid - Returns the connection ID for the Fserve.
.hwnd - Returns the window handle for the Fserve window.
.idle - Returns the number of seconds the connection has been idle.
.ip - The IP Address for the remote user.
.status - The status of the window; returns: active, inactive, or waiting.
.stamp - Returns either $true, or $false depending on if time-stamping is enabled or disabled.
.wid - Returns the window ID for the Fserve window.
.logfile - Returns the full logfile name if a log is opened for the window, otherwise returns $null.

Example

Echo all open Fserve connections: 
//echo -a $fserv(0)
Echo the current directory of the 1st Fserve: 
//echo -a $fserv(1).cd

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Depreciated: mIRC v6.3 (16 Aug 2007)
