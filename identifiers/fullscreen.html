$fullscreen

$fullscreen



The $fullscreen identifier returns '1' if mIRC is in fullscreen mode, '0' otherwise.

Synopsis

$fullscreen

Parameters

None

Properties

None

Example

Echo to the active window to see if fullscreen mode is enabled: 
//echo -a $fullscreen

Compatibility

Added: mIRC v6.3 (16 Aug 2007)

Notes

None
