$fullscreen



The $fullscreen identifier returns 1 if an application is currently running in full screen mode, 0 if not. This applies to both game and non-game applications.

Synopsis

$fullscreen

Paramters

None

Properties

None

Example

//echo -a $fullscreen

Compatibility

Added: mIRC v6.3 (16 Aug 2007)

Notes
