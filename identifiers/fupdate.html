$fupdate

$fupdate



The $fupdate identifier returns the current integer 0-100 setting for the /fupdate command

Synopsis

$fupdate

Parameters

None

Properties

None

Example

; save current /fupdate setting, then display mirc.ini to status window, then restore previously existing setting:
//var -s %a fupdate $fupdate | fupdate 99 | filter -fs mirc.ini * | fupdate %fupdate

Compatibility

Added: mIRC v7.65 (07 May 2021)

Notes

The /fupdate setting always reverts to default 0 each time mIRC restarts, so it must be reset each time using an ON START or perform-on-connect command
