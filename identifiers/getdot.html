$getdot



The '''$getdot''' identifier returns the RGB color value of a specific pixel from a picwin.

Synopsis

$getdot(picwin,x,y)

Parameters

{{ArgsList | picwin | The picwin of which to retrieve the RGB color value from | x | The x position within the picwin | y | The y position within the picwin }}
