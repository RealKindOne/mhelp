$group

$group



The $group identifier returns the name or status of a #group in a script.

Synopsis

$group(N/name)[.property]

Parameters

N - The Nth group
name -the name of a group, prefixed with a '#'

Properties

.status - Return the status of the group, on or off.
.name - Return the name of the group.
.fname - Return the filename in which the group exists.

Example

//echo -a $group(0)
//echo -a $group(1).name
//echo -a $group(#name).status
//echo -a $group(#name).fname

Compatibility

Added: mIRC v5.0 (02 Apr 1997)

v5.0 : Added .status property.
v5.4 : Added .fname and .name properties.
