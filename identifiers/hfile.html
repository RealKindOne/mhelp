$hfile



The $hfile identifier could be used in the past to display a selection file dialog just like the old $file='text dir' identifier, but would lay out the file in the dialog horizontally. Nowadays it returns the same $sfile.

Note: This feature is depreciated and has been replaced with $sfile.

Synopsis

$hfile="title" dir

Parameters

title - the title of the dialog, the quote are optional if you don't use space, this parameter is optional: //echo -a $hfile c:\*.txt
dir - the directory you want to display, you can put a file at the end which will be used to fill the 'filename' field in the dialog

Properties

None

Example

//echo -a > $hfile="select" c:\

Compatibility

Added: mIRC v4.1 (27 Apr 1996)

Notes

None
