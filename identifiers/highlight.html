$highlight



The $highlight identifier returns a $true/$false value depending on if you have the highlight features enabled in mIRC.

Synopsis

$highlight

Parameters

None

Properties

None

Example

Echo $true if highlighting is enabled or false if disabled: 
//echo -a $highlight

Compatibility

Added: mIRC v5.11 (11 Sep 1997)

Notes

None
