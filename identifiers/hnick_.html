$hnick()



The $hnick() identifier returns the Nth half-op nick in a channel.

Note: This feature is depreciated and has been replaced with $nick().

Synopsis

$hnick(#,N)

Parameters

None

Properties

None

Example

Echo total number of half-op nicks in #foobar: 
//echo -a $hnick(#foobar,0)
Echo 1st half-op nick in #foobar: 
//echo -a $hnick(#foobar,1)

Compatibility

Added: mIRC v5.4 (23 Jun 1998)

Depreciated: mIRC v5.6 (03 Jun 1999)

None
