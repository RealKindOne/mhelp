$hotline



{{Deprecated feature|new=$hotlink
}}The '''$hotline''' identifier returns the entire line of text that matched for an on_HotLink event.

Synopsis

$hotline

Parameters

None

Example

The below example will trigger when the mouse double-clicks on the word ''hoverme'', and echo the hotlink word and the entire line to the active window: 
ON ^*:HOTLINK:*hoverme*:*:{
  if ($1 == hoverme) return
  halt
}
ON *:HOTLINK:*hoverme*:*:echo -a Hotlink: $1 - Hotlink line: $hotline

Compatibility

Added: mIRC v5.61 (23 Sep 1999)
