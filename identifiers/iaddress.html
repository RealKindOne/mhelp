$iaddress



The $iaddress identifier returns the ip address, if avaible, inside an {{mIRC|on dns}} event.

Synopsis

$iaddress

Paramters

None

Properties

None

Example

on *:dns:echo -a $iaddress

Added: mIRC v4.7 (09 Dec 1996)
