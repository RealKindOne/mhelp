$iaddress

$iaddress



The $iaddress identifier returns the IP address, if available, inside an On_Dns event.

Note: This feature is depreciated and has been replaced with $dns().addr.

Synopsis

$iaddress

Parameters

None

Properties

None

Example

on *:dns:echo -a $iaddress

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

Depreciated: mIRC v6.0 (03 Feb 2002)
