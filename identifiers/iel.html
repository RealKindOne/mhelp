$iel



The $iel identifier returns the Nth address in the internal exception list. You must perform "/mode #channel e" to fill the internal exempt list.

Synopsis

$iel(#chan,N)[.property]

Parameters

#chan - The channel you want the exception from.
N - The Nth exception address in the channel, use N = 0 for the total number of addresses.

Properties

.by - Returns the nickname who set the exception.
.date - Returns the date when the exception was set.
.ctime - Returns the date when the exception was set, in $ctime format.

Example

Echo total numbers of exceptions for #mirc: 
//echo -a $iel(#mirc,0)

Compatibility

Added: mIRC v6.17 (17 Feb 2006)

v6.17: Added .by, .ctime, and .date properties.
