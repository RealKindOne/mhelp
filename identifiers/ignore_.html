$ignore()

$ignore()



The $ignore() identifier allows you to get data from your /ignore (Ignore) list in mIRC.

Synopsis

$ignore(address|N)[.property]

Parameters

address - Returns any match based on the specified address from the ignore list.
N - Returns any match in the auto-voice list based on the Nth entry in the list. If N is 0, the total amount of addresses in the ignore list will be returned.

Properties

.type - Returns the ignore flag for the ignored address.
.network - Returns the network associated with the ignored address.
.secs - Returns the number of seconds until ignore is removed if /ignore -uN were used.

Example

Echo to the active window the total number of addresses in the ignore list: 
//echo -a $ignore(0)

Compatibility

Added: mIRC v5.1 (28 Aug 1997)
