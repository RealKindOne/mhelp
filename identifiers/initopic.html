$initopic



The $initopic identifier returns the name/Nth position of the specified topic in an ini/text file.

Note: This feature is depreciated and has been replaced with $ini.

Synopsis

$initopic(file,topic/N)

Parameters

file - The filename you want name/position from.
topic/N - The Nth topic or a topic name, use N = 0 for the total number of topics.

Properties

None
