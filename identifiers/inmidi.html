$inmidi

$inmidi



The $inmidi identifier returns whether or not a midi file is currently playing.

The Note: This identifier is also a part of the Playing Music section.

Synopsis

$inmidi

Paramaters

None

Properties

.fname - Returns the file name of the currently playing midi file.
.pos - Returns the current position that the player is playing the midi at.
.length - Returns the full length of the currently playing midi file.
.pause - Returns $true if the currently playing midi file is paused, else it returns $false.

Example

'Echo whether or not a midi file is playing to the active window 
//echo -a $inmidi

Compatibility

Added: mIRC v5.0 (02 Apr 1997)
