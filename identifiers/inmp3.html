$inmp3



{{Deprecated feature|new=$insong
}}'''$inmp3''' return $true/$false if an mp3 is playing or information about the mp3 via some properties.
Note:This identifier is also a part of the Playing Music section.

Synopsis

$inmp3

Parameters

None

Properties

.fname - Returns the filename of the mp3 currently playing
.pos - Returns the position of the current mp3 playing
.length - Returns the length of the mp3.
.pause - Returns $true if an mp3 is currently paused, $false otherwise

Example

//echo -a $inmp3

Compatibility

Added: mIRC v5.8 (05 Sep 2000)
