$inrect



The $inrect identifier returns $true
if the specified point is inside the specified rectangle, $false
otherwise

Synopsis

$inrect(x1,y1,x,y,w,h)

Parameters

* '''x1''' - the x coordinate of the point * '''y1''' - the y coordinate of the point * '''x''' - the x coordinate of the rectangle * '''y''' - the y coordinate of the rectangle * '''w''' - the width of the rectangle * '''h''' - the height of the rectangle

Properties

None

Example

//echo -a $inrect(50,50,10,10,100,100)

Compatibility

Added: mIRC v5.3 (13 Dec 1997)

Notes

None
