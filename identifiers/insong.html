$insong



The $insong identifier returns whether or not an mp3, wma, or ogg file is currently playing.

Note: This command is also a part of the Playing Music section.

Synopsis

$insong

Paramaters

None

Properties


.fname - Returns the file name of the currently playing mp3, wma, or ogg file.
.pos - Returns the current position that the player is playing the mp3, wma, or ogg at.
.length - Returns the full length of the currently playing mp3, wma, or ogg file.
.pause - Returns $true if the currently playing mp3, wma, or ogg file is paused, else it returns $false.

Example

Echo whether or not a mp3, wma, or ogg file is playing to the active window 
//echo -a $insong

Compatibility

Added: mIRC v6.0 (03 Feb 2002)
