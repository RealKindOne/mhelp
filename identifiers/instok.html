$instok



The $instok identifier inserts token into the Nth position in text, even if it already exists in text.

Synopsis

$instok(<list>,token,N,C)

Parameters

<list> - The list of tokens (can be empty)
token - the token to insert
N - The Nth position to insert the token into
<C> - The ascii number (or code point) of the character separating the tokens If you pass $null for N, $null is returned. Otherwise, it's the Nth position to insert the token: -passing 0 will insert the token at the end -N can be negative, in this case the position it refers to starts from the end of the string instead. $instok(a b c d e,h,-2,32) = "a b c h d e"

Properties

None

Example

return one two three four 
$instok(one two three, four,4, 32)

Compatibility

Added: mIRC v4.7 (09 Dec 1996)
