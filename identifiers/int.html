$int



The $int identifier returns the integer part of a floating point number with no rounding.

Synopsis

$int(N)

Parameters

N - The number you want the integer part of.

You can use the letter 'e' or 'd' as *10^N, $int(5e6) = $int($calc(5*10^6))

Properties

None

Example

will return "1", the integer part 
//echo -a $int(1.9)

Compatibility

Added: mIRC v5.1 (28 Aug 1997)

Notes
