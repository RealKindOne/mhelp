$inwho



The $inwho is a value returned by {{mIRC|$chan|$chan().ial}} or $chan().inwho if mIRC is currently handling a /who reply and this is the first /who.

Synopsis

$inwho

Paramters

None

Properties

None

Example

//echo -a $chan().inwho
