$isadmin

$isadmin



The $isadmin identifier returns $true if mIRC is running in an elevated, administrator state, otherwise $false.

Synopsis

$isadmin

Parameters

None

Properties

None

Example

//echo -a $isadmin

Compatibility

Added: mIRC v7.58 (17 Nov 2019)

Notes

If mIRC is running in elevated 'run as administrator' state:
All executables launched with /run will also have elevated state, even if /run's -a switch is not used.
The UAC prompt will not appear when using /run -a to launch a program.
