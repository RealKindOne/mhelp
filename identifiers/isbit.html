$isbit

$isbit



The $isbit identifier returns 1 if the Nth bit is turned on

Synopsis

$isbit(A,N)

Parameters

A - The number, in base10/decimal
N - The Nth bit you want to check

Properties

None

Example

//var %a 5 | echo -a $base(%a,10,2) $isbit(%a,2)

Compatibility

Added: mIRC v5.61 (23 Sep 1999)

Notes

None
