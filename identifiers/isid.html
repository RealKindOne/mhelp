$isid

$isid



The $isid identifier returns $true if the custom alias is called as an identifier, $false otherwise.

Synopsis

$isid

Parameters

None

Properties

None

Example

/echo -a IS: $test /test 
alias test {
  if ($isid) return $true
  else echo -a $false
}

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
