$islower



The $islower identifier returns $true if text is all lower case, otherwise returns $false.

Synopsis

$islower(text)

Parameters

text - The input string to be checked

Properties

None

Example

Return $true 
//echo -a $islower(abcdefg)
Return $false because the letter G is uppercase. 
//echo -a $islower(abcdefG)

Compatibility

Added: mIRC v5.61 (23 Sep 1999)
