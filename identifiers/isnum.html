$isnum



The $isnum identifier returns $true if text is a number, otherwise returns $false. '''Note''': switches affects the range number parameters, see the examples

Synopsis

$isnum(text,[sd],[I,J])

Parameters

text - The input string to be checked

s - also permits +- sign to be used
d - also permits the decimal, allowing numbers with fractions
[I,J] - If you pass a third and fourth parameters, they must be valid number (with respect to the switches used) and are used to validate the number in a range.

Properties

None

Example

//echo -a $isnum(564)
will return '$true' 
//echo -a $isnum(abcdefG)
will return '$false' because not all the characters are digits //var %a -123 | echo -a $isnum(%a) vs $isnum(%a,s) result: $false vs $true (using a sign requires 's') //echo -a $isnum(3,d,2.1,4) vs $isnum(3,,2.1,4) result: $false vs error, the d switches is required not only to allow decimal in the input, but in each range parameter as well
