$isupper



The $isupper identifier returns $true if text is all upper case, otherwise returns $false.

Synopsis

$isupper(text)

Parameters


text - The string to be checked

Properties

None

Example

Return $true 
//echo -a $isupper(ABCDEFG)
Return $false because the letter g is lower vase. 
//echo -a $isupper(ABCDEFg)
