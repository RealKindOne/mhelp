$knick

$knick



The $knick identifier refers to the nickname of a user who has been kicked from a channel. This can also be the local mIRC client nickname if an [[On me - mIRC|ON ME]] TODO kick event is being monitored.

Synopsis

$knick

Parameters

None

Properties

None

Example

Send a notice to a kicked user with information about the kick: 
ON *:KICK:#: {
  .notice $knick You were kicked from $chan by $nick for the following reason: $iif($1-,$1-,No reason given)
}
The following is an example of the above code: You were kicked from #BreakingBad by WalterWhite for the following reason: Stop mass-highlighting!

Compatibility

Added: mIRC v2.1a (28 Feb 1995)
