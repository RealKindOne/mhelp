$lactive

$lactive



The $lactive identifier will return the full name of the last window that was active.

Synopsis

$lactive

Parameters

None

Properties

None

Example

Echo the value of $lactive to the active window 
//echo -a $lactive
Listen for an active window change, then echo the last active window 
ON *:ACTIVE:*:echo -a * Active window is now $active $+ . Previous active window was $lactive

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
