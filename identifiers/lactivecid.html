$lactivecid



The $lactivecid identifier returns the connection id related to the last active window. For instance if the you have more than one connection, let's say connections ''2'' and ''55'', and you activate a window in connection ''55'' and then switch back to connection ''2'', $lactivecid is now set to ''55''.

Synopsis

$lactivecid

Parameters

None

Properties

None

Example

Echo to the currently active window the connection ID of the last active window'''
