$len



The $len(text) identifier returns the number of characters in the text.

Synopsis

$len(text)

Parameters


text - The input string you want to know the length of

Properties

None

Example

Prints 3 
//echo -a $len(ONE)
Print 7 
//echo -a $len(ONE TWO)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
