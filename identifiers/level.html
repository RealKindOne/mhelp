$level

$level



The $level finds a matching address in the remote users list and returns its corresponding levels list.

Synopsis

$level(address)

Parameters

address - The address used you want the correspond levels list of

Properties

None

Example

//echo -a $level(*!*@mirc.com)
would display something like =5,10,20,21,32

Compatibility

Added: mIRC v7.0 (02 Apr 2010)

Notes

None
