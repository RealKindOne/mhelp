$lines



The $lines identifier returns the total number of lines in the specified text file.

Synopsis

$lines(filename)

Parameters

filename - The complete filename you want to know the total number of lines.

Properties

None

Example

Return the total number of lines in the relative filename readme.txt. 
//echo -a $lines(readme.txt)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
