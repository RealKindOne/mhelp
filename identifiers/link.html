$link

$link



The $link identifier returns the Nth item in the server Links window (called Links List), which can be retrieved with /links.

Synopsis

$link(N)[.property]

Parameters

N - The Nth link in the Links list window, if N is 0, returns the total number of links.

Properties

.addr - Returns the address of the server, same as $link(N) without property (default)
.ip - Returns the IP address associated with the server, if any
.level - Returns the level for that server
.info - Returns information about that server.

Example

//echo -a $link(0)

Compatibility

Added: mIRC v5.1 (28 Aug 1997)

v5.1 : Added .addr, .info, ip, and .level properties.

Notes

None
