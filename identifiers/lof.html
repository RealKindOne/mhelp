$lof



The $lof(filename) identifier returns the bytes size information about the specified file, same as $file().size identifier.

Synopsis

$lof(filename)

Parameters

Filename - the filename that you want to check for

Properties

None

Example

Return the bytes size of the mirc.exe file that you already using. 
//echo -a $lof($mircexe)
Return the bytes size (e.g: 14056) of the mirc.ini file that you already using. 
//echo -a $lof(mirc.ini)
Return the original size (e.g: 14KB) of the mirc.ini file that you already using. 
//echo -a $bytes($lof($mircini)).suf

Compatibility

Added: mIRC v4.52 (06 Jul 1996)

Notes

It will return $null if the file doesn't exists to the path.
