$log2



The $log2 identifier returns the common logarithm (base 2) of a number.

Synopsis

$log2(N)

Parameters

N - Number for which to return its common logarithm.

Properties

None

Example

//echo -a except for precision rounding 2^ $log2(7) is 7
//echo -a $log2(7) is $log(7) / $log(2)

Compatibility

Added: mIRC v7.72 (27 Nov 2022)

Notes

None
