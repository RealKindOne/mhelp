$longip



The $longip identifier converts an IP address into a long value and vice-versa.

Synopsis

$longip(ip)

Parameters


ip - The ip address to convert, if you pass a long value, it will return the ip address back

Properties

None

Example

//echo -a $longip(158.152.50.239) -- $longip(2660774639)

Compatibility

Added: mIRC v4.6 (07 Sep 1996)

Notes

None
