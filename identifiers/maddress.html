$maddress

$maddress



The $maddress identifier return the address that was matched for the event, or returns information about the user list.

Note: This feature is deprecated and has been replaced with $ulist.

Synopsis

$maddress

Parameters

None

Properties

None

Example

To get the address from the event: 
on *:TEXT:*:#:echo -a $maddress

Compatibility

Added: mIRC v4.6 (07 Sep 1996)

Notes

Deprecated: mIRC v6.0 (03 Feb 2002)
