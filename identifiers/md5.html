$md5

$md5



The $md5identifier calculates the MD5 hash of a text, binary_variables, or file.

Synopsis

$md5(text|&binary|filename,[N])

Parameters

text - Used with N = 0, just the string you want the md5 hash of.
&binary - A binary variable, used with N = 1, returns the md5 hash of the content of the binvar.
filename - A filename, used with N = 2, returns the md5 hash of the content of the file.

Properties

None

Example

//echo -a $md5($mircexe,2)

Compatibility

Added: mIRC v6.03 (16 Aug 2002)
