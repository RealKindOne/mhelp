$menu



The $menu identifier return the name of menu in use.

Synopsis

$menu

Parameters

None

Properties

None

Example

menu nicklist,channel {
  $iif($menu == nicklist,...):{}
  $iif($menu == channel,...):{}
}

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
