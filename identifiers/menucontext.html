$menucontext

$menucontext



The $menucontext identifier return the context in which the popups is created. The value can be 'window' or 'treebar'.

Synopsis

$menucontext

Parameters

None

Properties

None

Example

menu nicklist,channel, {
  $iif($menu == nicklist,...):{}
  $iif($menu == channel && $menucontext == treebar,...):{}
}

Compatibility

Added: mIRC v6.3 (16 Aug 2007)

Notes

None
