$mididir



The $mididir identifier returns the directory specified in the Sound Requests section of the Options dialog for midi files

Note: This feature is deprecated and has been replaced by $sound(midi)

Synopsis

$mididir

Parameters

None

Properties

None

Example

//echo -a $mididir

Compatibility

Added: mIRC v5.0 (02 Apr 1997)

Notes

Depreciated: mIRC v6.0 (03 Feb 2002)
