$mircdir

$mircdir



The $mircdir identifier returns the folder in which mIRC stores its main settings files such as mirc.ini.

Synopsis

$mircdir

Parameters

None

Properties

None

Example

//echo -a $mircdir

Compatibility

Added: mIRC v4.52 (06 Jul 1996)

Notes

None
