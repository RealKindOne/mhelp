$mircexe



The $mircexe identifier returns the complete filename to the executable currently used by the application.

Synopsis

$mircexe

Parameters

None

Properties

None

Example

Echo the complete filename: 
//echo -a $mircexe

Compatibility

Added: mIRC v5.5 (08 Jan 1999)

Notes

None
