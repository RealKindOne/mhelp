$mircini



The $mircini identifier return the complete filename to the mirc.ini setting file currently used by the application.

Synopsis

$mircini

Parameters

None

Properties

None

Example

//echo -a $mircini

Compatibility

Added: mIRC v5.02 (21 Apr 1997)
