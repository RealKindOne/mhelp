$mklogfn



The $mklogfn identifier returns a the filename that would be used for such a nickname as a log file, it ensures a valid filename by doing the same underscores replacement as $mkfn, and then applying your log setting (adding a network and .log etc)

Synopsis

$mklogfn(text)

Parameters

text - Text to be made a valid log filename respecting your mIRC log settings

Properties

None

Example

//var %i 33 | var %f | while (%i isnum 33-126) { var %f %f $+ $chr(%i) | inc %i } | echo -a orig: %f | echo -a mklogfn: $mklogfn(%f)

Compatibility

Added: mIRC v5.7 (02 Feb 2000)

Notes

While this ensures a string is a valid filename, it does not ensure 2 different strings wouldn't get assigned the same filename, with both having different characters replaced with the same underscore.
