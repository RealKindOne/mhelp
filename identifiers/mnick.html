$mnick



The $mnick identifier returns the value of your main nickname, which is set in the mIRC options dialog.

Synopsis

$mnick

Parameters

None

Properties

None

Example

Echo the value of $mnick to the active window: 
//echo -a $mnick

Compatibility

Added: mIRC v5.8 (05 Sep 2000)

Notes

None
