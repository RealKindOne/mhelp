$modefirst



The $modefirst identifier returns $true if the channel event is the first to trigger, $false otherwise.

Synopsis

$modefirst

Parameters

None

Properties

None

Example

send /mode #channel +oo nick1 nick2 
on *:op:#:{
  echo -a $modefirst
}

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
