$modelast



The $modelast identifier returns $true if the channel event is the last to trigger, $false otherwise.

Synopsis

$modelast

Parameters

None

Properties

None

Example

send /mode #channel +oo nick1 nick2 
on *:op:#:{
  echo -a $modelast
}

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
