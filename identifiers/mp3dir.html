$mp3dir



The $md3dir identifier returns the directory specified in the Sound Requests section of the Options dialog for mp3 files.

Note: This feature is deprecated and has been replaced by $sound(mp3)

Synopsis

$mp3dir

Parameters

None

Properties

None

Example

//echo -a $mp3dir

Added: mIRC v5.8 (05 Sep 2000)
