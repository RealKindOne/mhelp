$msgstamp

$msgstamp



The $msgstamp identifier returns UTC timestamp for a irc server message that has an IRCv3 @time prefix tag, such as when CAP server-time or znc.in/server-time[-iso] are enabled on a server.

Synopsis

$msgstamp

Parameters

None

Properties

None

Example

Reuse a ZNC playback buffer message timestamp with a self-defined format: 
//echo -a $asctime($msgstamp, "[HH:nn:ss]") $text

Compatibility

Added: mIRC v7.33 (27 May 2014)
