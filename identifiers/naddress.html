$naddress

$naddress



The $naddress identifier returns the address being resolved in an On_Dns event.

Note: This feature is depreciated and has been replaced with $dns().addr.

Synopsis

$naddress

Parameters

None

Properties

None

Example

on *:dns:{
  echo -a $naddress
}

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

Depreciated: mIRC v6.0 (03 Feb 2002)
