$no



The $no identifier is returned when you invoke an $input box with the '''v''' switch and the '''No''' button was clicked.

Synopsis

$no

Parameters

None

Example

Creates an input with a no button that will echo results to the active window: 
//echo -a $input(Click no to see the result.,yv)

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

Notes

None
