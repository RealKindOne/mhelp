$nofile



The $nofile identifier returns the directory path in a full filename without the name of the file itself.

Synopsis

$nofile(full-filename)

Parameters

full-filename - The filename you want the path of.

Properties

None

Example

Echo the directory of the mIRC executable that is running: 
//echo -a $nofile($mircexe)

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

None
