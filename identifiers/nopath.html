$nopath

$nopath



The $nopath identifier returns filename without a path if it has one.

Synopsis

$nopath(filename)

Parameters

filename - the filename you want the path of

Properties

None

Example

Print "mirc.exe" filename without the path. 
//echo -a $nopath($mircexe)

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

None
