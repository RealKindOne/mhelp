$notags



The $notags identifier returns the text stripped from all the possible tags for spoken text in /gtalk

Synopsis

$notags(text)

Parameters

text - the text containing tags for spoken text

Properties

None

Example

//echo -a $notags(te\spd=5\st)

Compatibility

Added: mIRC v5.7 (02 Feb 2000)

Notes

None

See also

/gtalk
