$ok

$ok



The $ok identifier is returned when $input box has been invoked with the the v switch and the Ok button has been pressed.

Synopsis

$ok

Parameters

None

Properties

None

Example

Creates an input with an ok button that will echo results to the active window: 
//echo -a $input(Click ok to see the result.,ov)

Compatibility

Added: mIRC v5.9 (26 Apr 2001)
