$online

$online



The $online identifier return the total number of seconds in the Timer dialog (ALT + I).

Synopsis

$online

Parameters

None

Properties

None

Example

Echo the total number of seconds mIRC has been connected: 
//echo -a $online
Message a channel when someone types "!uptime": 
on *:text:!uptime:#example:{
  msg $chan My grand total connection time has been $online seconds.
  msg $chan My current connection time has been $onlineserver seconds.
  ; /!\ Always use 'n' for $readini()  /!\
  msg $chan The total was reset at: $readini($mircini,n,text,lastreset)
}

Compatibility

Added: mIRC v4.72 (09 Jan 1997)

Notes

This is global, it only requires one active connection to increase the total.

The $onlinetotal identifier does the same thing as this identifier.
