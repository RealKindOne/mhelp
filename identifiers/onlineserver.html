$onlineserver

$onlineserver



The $onlineserver identifier returns the current number of seconds for the server connection time in the Timer dialog (ALT + I).

Synopsis

$onlineserver

Parameters

None

Properties

None

Example

Echo the total number of seconds mIRC has been connected for the current session: 
//echo -a $onlineserver
Message a channel when someone types "!uptime": 
on *:text:!uptime:#example:{
  msg $chan I have currently been connected $onlineserver seconds 
  msg $chan I connected at: $asctime($calc($ctime - $onlineserver))
}

Compatibility

Added: mIRC v7.35 (04 Aug 2014)

Notes

This only applies for the current server the identifier was triggered on.
