$onlinetotal

$onlinetotal



The $onlinetotal identifier returns the total number of seconds in the Timer dialog (ALT + I).

Synopsis

$onlinetotal

Parameters

None

Properties

None

Example

Echo the total number of seconds mIRC has been connected: 
//echo -a $onlinetotal
Message a channel when someone types "!uptime": 
on *:text:!uptime:#example:{
  msg $chan My grand total connection time has been $onlinetotal seconds.
  msg $chan My current connection time has been $onlineserver seconds.
  ; /!\ Always use 'n' for $readini()  /!\
  msg $chan The total was reset at: $readini($mircini,n,text,lastreset)
}

Compatibility

Added: mIRC v7.35 (04 Aug 2014)

Notes

This is global, it only requires one active connection to increase the total.

The $online identifier does the same thing as this identifier.
