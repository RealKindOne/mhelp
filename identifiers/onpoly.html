$onpoly



The $onpoly identifier returns $true
if two polygon overlap, $false
otherwise

Synopsis

$onpoly(n1,n2,x1,y1,xN1,yN1,x2,y2,xN2,yN2,...)

Parameters

* '''n1''' - the number of points defining the first polygon * '''n2''' - the number of points defining the second polygon * '''x1''' - the x coordinate of the first point for the first polygon * '''y1''' - the y coordinate of the first point for the first polygon * '''xN1''' - the x coordinate of the Nth point for the first polygon * '''yN1''' - the y coordinate of the Nth point for the first polygon * '''x2''' - the x coordinate of the first point for the second polygon * '''y2''' - the y coordinate of the first point for the second polygon * '''xN2''' - the x coordinate of the Nth point for the second polygon * '''yN2''' - the y coordinate of the Nth point for the second polygon
