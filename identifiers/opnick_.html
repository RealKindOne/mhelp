$opnick()



The $opnick() identifier returns the Nth oped nick in a channel.

Note: This feature is depreciated and has been replaced with $nick().

Synopsis

$opnick(#,N)

Parameters

None

Properties

None

Example

Echo total number of oped nicks in #foobar: 
//echo -a $opnick(#foobar,0)
Echo 1st oped nick in #foobar: 
//echo -a $opnick(#foobar,1)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Depreciated: mIRC v5.6 (03 Jun 1999)

Notes

None
