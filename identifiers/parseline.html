$parseline

$parseline



The $parseline identifier returns the line involved in the On_parselineevent.

Synopsis

$parseline

Note: For "out" lines, $parseline is terminated with a $LF character that is not visible when viewed in mIRC. You may wish to remove this with: 
var %pl $parseline
if ($asc($right(%pl,1)) == 10) %pl = $right(%pl,-1)

Parameters

None

Properties

None

Example

on *:parseline:*:*:echo -a $parseline

Compatibility

Added: mIRC v7.42 (17 Jul 2015)
