$parseutf

$parseutf



The $parseutf identifier returns $true if mIRC is going to utf encode/decode during the on_parseline event, $false otherwise.

Synopsis

$parseutf

Parameters

None

Properties

None

Example

on *:parseline:*:{
  echo -a $parseutf
}

Compatibility

Added: mIRC v7.42 (17 Jul 2015)

Notes

None
