$passivedcc



The $passivedcc identifier return $true if the passive dcc option is enabled in mIRC, $false otherwise.
The DCC PASSIVE option can be enabled or disabled either in Alt-O/Dcc/Options "Passive DCC's" or with "/dcc passive ON" or "/dcc passive OFF".

Synopsis

$passivedcc

Parameters

None

Properties

None

Example

//echo -a $passivedcc
