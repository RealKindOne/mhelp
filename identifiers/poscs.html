$poscs



The $poscs identifier returns the case-sensitive position of the Nth substring within a text string. $pos is the case-insensitive version.

Synopsis

$poscs(text,substring [,N] )

Parameters

{{ArgsList | %var/text | The string being searched | substring | The string being searched for within text string | N | The Nth occurrence of substring being searched for. Default is 1 if parameter not used. }} * Note: N=0 returns count of string's occurrences. Returns $null if Nth substring not found. == Switches == None
