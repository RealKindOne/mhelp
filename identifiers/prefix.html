$prefix



The $prefix identifier returns the list of channel user nickname prefixes (to indicate channel op, voice etc. status) supported on the active network connection. If not connected, mIRC defaults to a standard value of (ohv)@%+.

Parameters

None

Properties

None

Example

Echo the current available channel user nickname prefixes to the active window: 
//echo -a $prefix

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

Notes

Any active connection window can use this identifier, not just an active channel.
