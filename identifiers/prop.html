$prop



The $prop identifier returns the name of the property passed to your custom alias, if any.

Synopsis

$prop

Parameters

None

Properties

None

Example

alias test {
  echo -a $prop
}
//noop $test().prop

Compatibility

Added: mIRC v5.7 (02 Feb 2000)
