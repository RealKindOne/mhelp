$protect

$protect



The $protect identifier returns the status of auto-protect, $true for enabled, $false for disabled.

Synopsis

$protect

Parameters

None

Properties

None

Example

Echo to the active window whether or not auto-protect is enabled: 
//echo -a $protect

Compatibility

Added: mIRC v5.6 (03 Jun 1999)
