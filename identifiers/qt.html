$qt



The $qt identifier returns the text string enclosed with double-quotes.

Synopsis

$qt(<TEXT>)

Parameters

TEXT - Text string to ensure is enclosed in double quotes.

Properties

None

Example

Does not affect double-quotes within the text, only adds a double-quote to the start/end of TEXT if a double-quote is not already present. 
//var %ini $qt($mircini) | echo -a %ini | echo -a $qt(%ini)
; returns path\filename of your mirc.ini with double-quotes around it.
; no change in 2nd line because quotes already present
//echo -a $qt("""Test""String) ; returns: """Test""String" ; No change to internal quotes, no alteration of string start because at least 1 quote already present, only added missing quote to end of string.

Compatibility

Added: mIRC v6.17 (17 Feb 2006)

Notes

None
