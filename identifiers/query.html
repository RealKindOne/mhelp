$query

$query



The $query identifier returns the name of the Nth query window opened.

Synopsis

$query(N/nick)[.property]

Parameters

N - The Nth query window, if N = 0, returns the total number of query window opened.
nick - If a nickname is passed, returns the nickname if you have a query window with that nick, $null otherwise.

Properties

.addr - Returns the user address, this value might not be correct, you have to get a message from the user to make sure the address is correct
.logfile - Returns the full logfile name if a log is opened for the window, otherwise returns $null.
.stamp - Returns either $true, or $false depending on if time-stamping is enabled or disabled.
.wid - Returns the window ID of that query window.
.cid - Returns the connection ID for the query.
.hwnd - Returns the window handle for the query window.
.idle - Returns the number of seconds since the query window opened or the person said something, whichever happened last.

Example

Print the total number of open query windows for the network. 
//echo -a $query(0)
