$raddress

$raddress



The $raddress identifier returns the resolved address of a /dns request in an On_Dns event.

Note: This feature is deprecated and has been replaced by $dns().addr

Synopsis

$raddress

Parameters

None

Properties

None

Example

on *:dns:{
  echo -a $raddress
}

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

None
