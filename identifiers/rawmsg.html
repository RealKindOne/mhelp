$rawmsg



The $rawmsg identifier return the raw server line inside an event generated by an IRC server, after mIRC decoded (utf8).
This can be pretty useful if you need to get the spaces preserved, which are lost in $1-

Synopsis

$rawmsg

Parameters

None

Properties

None

Example

on *:TEXT:*:#:echo -a $rawmsg
