$readini



The $readini identifier returns the associated value of the item name for the given topic in an INI structure disk file

WARNING: This identifier can be dangerous. Please use the "n" switch.
More information here.


Synopsis

$readini(filename, np, topic, item)
Legacy syntax: 
$readini [switches] topic item

Parameters

filename - The file from which text is returned
np - Switches, 'n' = non evaluation of line, 'p' = treats command pipe separator as code instead of plain text, same as $read switches

Example

Compatibility

Notes:
