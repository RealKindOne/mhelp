$readn

$readn



The $readn identifier returns the number of the line matched by the last $read attempt.

Synopsis

$readn

Parameters

None

Properties

None

Example

//noop $read($mircini,tnw,*mirc*) | echo -a $readn

Compatibility

Added: mIRC v5.4 (23 Jun 1998)

Notes

The $readn value remains visible to all scripts and events until the next $read. If the read attempt fails to match, it returns 0.

See also
