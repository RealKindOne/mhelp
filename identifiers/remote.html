$remote

$remote



The $remote identifier returns a bitwise integer indicating if CTCPs/Events/Raw are enabled.

Synopsis

$remote
The value are OR'ed together 1 | 2 | 4

0 is returned when remote are off, 7 when everything is enabled, see the example to check one of them manually.

Parameters

None

Properties

None

Example

if ($remote & 0) echo -a CTCPs, Events, and Raw are disabled.
if ($remote & 1) echo -a Only CTCPs is enabled.
if ($remote & 2) echo -a Only Events is enabled.
if ($remote & 3) echo -a Raw is disabled, CTCPs and Events are enabled.
if ($remote & 4) echo -a Only Raw is enabled.
if ($remote & 5) echo -a Events are disabled, CTCPs and Raw are enabled.
if ($remote & 6) echo -a CTCPs are disabled, Events and Raw are enabled.
if ($remote & 7) echo -a CTCPs, Events, and Raw are enabled.
