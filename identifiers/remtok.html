$remtok



The $remtok identifier removes the Nth matching token in text. $remtokcs is a case sensitive equivalent.

Synopsis

$remtok(<list>,token,N,C)

Parameters

<list> - The list of tokens (can be empty)
token - the token used to match
N - The Nth matching token, default to 1, if N is 0, applies to all matching tokens. Cannot be negative, if greater than number of token, return the original string. This parameter can be ommited entirely.
<C> - The ascii number (or code point) of the character separating the tokens

Properties

None

Example

Returns one three 
$remtok(one two three, two, 32)

Compatibility

Added: mIRC v4.7 (09 Dec 1996)
