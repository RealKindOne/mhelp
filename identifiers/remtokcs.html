$remtokcs



The $remtokcs identifier removes the Nth matching token in text using case sensitivity.

Synopsis

$remtokcs(<list>,token,N,C)

Parameters

<list> - The list of tokens (can be empty).
token - The token used to match.
N - The Nth matching token, default to 1, if N is 0, applies to all matching tokens. Cannot be negative, if greater than number of token, return the original string. This parameter can be omitted entirely.
<C> - The ascii number (or code point) of the character separating the tokens.

Properties

None

Example

Returns one two three 
$remtokcs(one two three twO, twO, 32)

Compatibility

Added: mIRC v5.91 (15 Jun 2001)
