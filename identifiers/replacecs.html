$replace



The $replacecs identifier performs search/replace on text strings. This is case sensitive $replacecs can replace an already replaced string, it searches all substrings at every position in the text and start from the beginning of the string whenever there is a replacement made, {{mIRC|$replacex}} can be used to prevent what has already been replaced from being checked again

Synopsis

$replacecs(text , substring, replace, stringN, replaceN)

Parameters

text - The string you want to replace from
substring - The 1st string you want to search for
replace - the replacement for the 1st string
substringN - The Nth string you want to search for
replaceN - the replacement for the Nth string '''Note''': Excluding the first parameter (input string), there must be an even number of substring/replace parameters
