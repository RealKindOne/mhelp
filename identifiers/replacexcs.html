$replacexcs



The $replacexcs Performs search/replace on text strings, but doesn't replace what's already been replaced. $replacex is the case-insensitive version. Note: $replace search each substring separately, $replace(1234,234,z,123,u) is 1z, "234" was first checked against "1234" and then "123" isn't even checked because we reached the end of the string. But $replacexcs does not work like that, $replacexcs(1234,234,z,123,u) is u4, because the main loop is done on the input string, trying for each position to match any of the substring, so 123 is found first.

Synopsis

$replacexcs(text , substring, replace, stringN, replaceN)

Parameters

text - The string you want to replace from
substring - The 1st string you want to search for
replace - the replacement for the 1st string
substringN - The Nth string you want to search for
replaceN - the replacement for the Nth string

Note: Excluding the first parameter (input string), there must be an even number of substring/replace parameters

Properties

None

Example

//echo -a $replacexcs(abc,ab,c,cc,d) vs $replace(abc,ab,c,cc,d)

Compatibility

Added: mIRC v6.15 (03 Jun 2004)
