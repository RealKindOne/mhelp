$reptok



The $reptok identifier replaces the Nth matching token in text with a new token. $reptokcs is a case sensitive version

Synopsis

$reptok(<list>,token,new,N,C)

Parameters

<list> - The list of tokens (can be empty)
token - the token used to match
new - the token used to replace the matching token.
N - The Nth matching token, default to 1, if N is 0, applies to all matching tokens. Cannot be negative. If N is greater than the number of token, original input is returned.
<C> - The ascii number (or code point) of the character separating the tokens

Properties

None

Example

$reptok(one two three, two, four, 32) returns one four three

Compatibility

Added: mIRC v4.7 (09 Dec 1996)
