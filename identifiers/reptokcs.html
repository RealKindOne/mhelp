$reptokcs



The $reptokcs identifier replaces the Nth matching token in text with a new token using case sensitivity. If N = 0, applies to all matching items.

Synopsis

$reptokcs(<list>,token,new,N,C)

Parameters


<list> - The list of tokens (can be empty)
token - the token used to match
new - the token used to replace the matching token.
N - The Nth matching token, default to 1, if N is 0, applies to all matching tokens.
<C> - The ascii number (or code point) of the character seperating the tokens

Properties

None

Example

Returns one two three four 
$reptokcs(one two three twO, twO, four, 32)

Compatibility

Added: mIRC v5.91 (15 Jun 2001)
