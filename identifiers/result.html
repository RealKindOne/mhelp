$result

$result



The $result identifier returns the value passed to the last /return (or /returnex) command only if the alias was called as a command (as of 7.51, may be extended to any aliases call in the future).

Synopsis

$result

Parameters

None

Properties

None

Example

alias test {
  return 5
}
alias testing {
  test
  echo -a $result
}

Compatibility

Added: mIRC v4.52 (06 Jul 1996)

Notes

None
