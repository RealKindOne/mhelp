$right



The $right identifier is a String manipulation identifier used to return characters from the right side of a string.

Synopsis

$right(<string>, <length>)

Parameters

<string> - The string to manipulate.
<length> - The length or number of characters to remove from the original string. <length> can be a positive or negative. Using 0 returns $null (0 characters).
If the length provided is positive, $right will return the <length> characters from the right of the string.

Properties

None

Example

Echo "defg": 
$right(abcdefg, 4)
If the length provided is negative, $right will return the characters in the string minus <length> on the left.
Echo "cdefg": 
$right(abcdefg, -2)
Echo "eseed": 
$right(appleseed, -4)
Echo "Name": 
$right([AB]BotName, 4)

Compatibility
