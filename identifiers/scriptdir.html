$scriptdir

$scriptdir



The $scriptdir identifier returns the folder containing the current script file, or if used in an editbox, return the default folder for script.

Synopsis

$scriptdir

Parameters

None

Properties

None

Example

//echo -a $scriptdir

Added: mIRC v5.0 (02 Apr 1997)
