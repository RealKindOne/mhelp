$serverip



The $serverip identifier returns the ip address of the IRC server you are connected to.
Since mIRC 7.1+, $serverip is also filled by /dns $server for SOCK5 users, but note that this result in a DNS leak.

Synopsis

$serverip

Parameters

None

Properties

None

Example

//echo -a $serverip

Compatibility

Added: mIRC v6.03 (16 Aug 2002)

Notes

None
