$servertarget



The $servertarget identifier returns the original address used to connect to the IRC server. This can be useful when dealing with BNC or Twitch.

Synopsis

$servertarget

Parameters

None

Properties

None

Example

//echo -a $servertarget

Added: mIRC v7.34 (18 Jun 2014)
