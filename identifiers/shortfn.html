$shortfn

$shortfn



The $shortfn(path) identifier returns short version of a long filename.

Synopsis

$shortfn(path)

Parameters

Path - The direction/path that you want to check for

Properties

None

Example

Return for example the "C:\Users\westor\DOCUME~1\mIRC\" short path version of the specified direction: 
//echo -a $shortfn($longfn($mircdir))
Return "C:\Users\westor\DOCUME~1\mIRC\" short path version of the long version one that you have specified: 
//echo -a $shortfn(C:\Users\westor\Documents\mIRC\)

Note: In order to return the short version of the path, the path MUST exist.

Compatibility

Added: mIRC v5.7 (02 Feb 2000)

Notes

None
