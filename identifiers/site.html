$site

$site



The $site identifier returns the portion of $address after the @ for the user associated with an event in the form user@host.

Synopsis

$site

Parameters

None

Properties

None

Example

on *:TEXT:*:#:{
  echo -a $site
}

Compatibility

Added: mIRC v3.9 (28 Dec 1995)

Notes

None
