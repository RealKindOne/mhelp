$sline



The $sline identifier returns the Nth selected line in a listbox.

Synopsis

$sline(win,N)[.property]

Parameters

win - The window with a listbox, can be a channel window or a custom window
N - The Nth selected line in the window

Properties

.ln - returns the line number of the Nth selected line

Example

//echo -a $sline(0)

Compatibility

Added: mIRC v5.0 (02 Apr 1997)

Notes

None
