$snick



The $snick identifier returns the Nth selected nickname on the specified channels nicklist.

Synopsis

$snick(#chan,N)

Parameters

#chan - The channel name you want to know the selected nickname from.
N - The Nth selected nickname on that channel.

Properties

None

Example

Print the total number of selected nicks for a channel: 
//echo -a $snick(#chan,0)
Print the second nick selected for a channel: 
//echo -a $snick(#chan,2)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

To clear the selected nicks you can use: 
/sline -r #chan
