$snick



The $snick identifier returns the Nth selected nickname on the specified channel

Synopsis

$snick(#chan,N)

Parameters

#chan - The channel name you want to know the selected nickname from
N - The Nth selected nickname on that channel.

Properties

None

Example

//echo -a $snick(0)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Notes

None
