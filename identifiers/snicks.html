$snicks

$snicks



The $snicks identifier returns a string of the currently selected nicknames in the active channel listbox, comma separated.

Synopsis

$snicks

Parameters

None

Properties

None

Example

//echo -a $snicks

Compatibility

Added: mIRC v3.7 (12 Oct 1995)

Notes

To clear the selected nicks you can use: 
/sline -r #chan

See also
