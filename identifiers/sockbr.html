$sockbr



The $sockbr identifier returns the number of bytes read from the last /sockread command done for that socket.

Synopsis

$sockbr

Parameters

None

Properties

None

Example

on *:sockread:name:{
  if ($sockerr) return
  var %a
  ;sockread %a reads one line, or nothing
  ;we check if we read anything before comparing
  if (!$sockbr) return
  if (%a == something) echo -a ok
}

Compatibility

Added: mIRC v5.3 (13 Dec 1997)
