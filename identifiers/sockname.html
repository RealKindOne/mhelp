$sockname

$sockname



The $sockname identifier returns the name given to a connection to identify it. This identifier can be used in events to know which connection an event is related to.

Synopsis

$sockname

Parameters

None

Properties

None

Example

on *:sockread:*:echo -a $sockname

Compatibility

Added: mIRC v5.3 (13 Dec 1997)

Notes

None
