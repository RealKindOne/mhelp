$speak



The $speak identifier returns the Nth line currently queued via the /speak command.

Synopsis

$speak(N)

Parameters


N - The Nth line queued, if N is 0, returns the total number of queued line.

Properties

None

Example

//echo -a $speak(0)

Compatibility

Added: mIRC v7.0 (02 Apr 2010)

Notes

None
