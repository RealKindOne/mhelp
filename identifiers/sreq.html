$sreq



The $sreq identifier returns the current parameter settings for DCC Sends for mIRC.

Synopsis

$sreq

Paramaters

None

Properties

None == Results == '''ask'''DCC Send requests will popup an authorization window asking you whether you wish to accept or decline all incoming DCC Sends. '''auto'''All incoming DCC Sends will be automatically accepted. '''ignore'''All incoming DCC Sends will be ignored.

Example

Echo the current status of DCC Sends to the active window: 
//echo -a $sreq

Compatibility

Added: mIRC v5.4 (23 Jun 1998)}

Notes

None

See also

/sreq
