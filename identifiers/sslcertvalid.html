$sslcertvalid

$sslcertvalid



The $sslcertvalid identifier returns $true if the IRC network's SSL certificate is valid for the current SSL connection. Returns $false if not connected or connection is non-SSL.

Synopsis

$sslcertvalid

Parameters

None

Properties

None

Example

//if (($server) && (+* iswm $port)) echo -a current SSL connection is $iif($sslcertvalid,valid,invalid)

Compatibility

Added: mIRC v7.58 (17 Nov 2019)
