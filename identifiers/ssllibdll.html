$ssllibdll

$ssllibdll



The $ssllibdll identifier returns the complete filename to the lib used for SSL. In recent versions of mIRC, SSL is integrated with mIRC and the $mircexe value will be returned.

Synopsis

$ssllibdll

Parameters

None

Properties

None

Example

//echo -a $ssllibdll

Compatibility

Added: mIRC v7.31 (26 Apr 2013)
