$sslready

$sslready



The $sslready identifier returns $true if you are capable of using SSL, $false otherwise.

If you are unable to use SSL, makes sure you have mirc correctly configured for SSL, see https://www.mirc.com/ssl.html for more information.

Synopsis

$sslready

Parameters

None

Properties

None

Example
