$starting

$starting



The $starting identifier return 1 in the on_Start event if mIRC is really starting, (on start could trigger after a /load command), 0 otherwise.

Synopsis

$starting

Parameters

None

Properties

None

Example

on *:start:{
  echo -s $starting
}

Compatibility

Added: mIRC v7.23 (19 Mar 2012)

Notes

None
