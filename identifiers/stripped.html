$stripped



The $stripped identifier returns the number of control codes that were stripped from an incoming message, if any, due to the settings in mIRC-options/IRC/Messages/"Strip codes from incoming messages". This can be used in any script event that triggers when a message is received from a user, including on_Text, on_action, and ctcp.

Synopsis

$stripped

Parameters

None

Properties

None

Example

on *:TEXT:*:#:echo -a $stripped

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

Notes

It does NOT count control codes stripped by the $strip identifier.
