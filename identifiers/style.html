$style



The $style can be used inside menu definition to create a checked or disabled (or both) menu item.

Synopsis

$style(N)

Parameters


N - use N = 1 for checked, N = 2 for disabled and N = 3 for both

Properties

None

Example

menu status {
  $iif($server == $null,$style(2)) Server Info
  .$style(1) This line is always checked:halt
  .Motd:/motd
  .Time:/time
}

Compatibility

Added: mIRC v5.9 (26 Apr 2001)

Notes

The $style(N) identifier must be the first word in the menu definition.
