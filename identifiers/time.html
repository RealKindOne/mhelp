$time

$time



The $time identifier returns the current time in hour:minute:second format. The time is returned in 24-hour clock format, not 12-hour clock format.

Synopsis

$time

Parameters

None

Properties

None

Example

Echo the current time to the active window: 
//echo -a $time

Compatibility

Added: mIRC v3.5 (07 Aug 1995)

Notes

None
