$toolbar



The $toolbar identifier returns 1 if the toolbar is enabled in mIRC, 0 otherwise.

Synopsis

$toolbar

Parameters

None

Properties

None

Example

Echo to see if the toolbar is enabled: 
//echo -a $toolbar

Compatibility

Added: mIRC v6.32 (21 May 2008)

Notes

None
