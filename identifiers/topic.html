$topic



The $topic identifier used to return the topic of the channel specified, this identifier has been removed from mIRC in version 4.7. Use $chan().topic instead.

Synopsis

$topic(<channel>)

Parameters

<channel> - The channel you want the topic of

Properties

None

Example

//echo -a $topic(#mirc)

Compatibility

Added: mIRC v4.6 (07 Sep 1996)
Removed: mIRC v4.7 (09 Dec 1996)

Notes

None

See also
