$treebar



The $treebar identifier gives you the current state of the mIRC treebar.

If the treebar is enabled you will get "1".
If the treebar is deabled you will get "0".

Synopsis

$treebar

Parameters

None

Properties

None

Example

Echo the treebar status to the active window: 
//echo -a $treebar

Compatibility

Added: mIRC v6.32 (21 May 2008)

None
