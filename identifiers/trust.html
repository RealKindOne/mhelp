$trust



The $trust identifier returns items in the dcc trust list.

Synopsis

$trust(N)

Parameters

N - The Nth item in the dcc trust list, if N = 0, returns the total number of item in the list

Properties

None

Example

//echo -a $trust(0)

Compatibility

Added: mIRC v6.01 (10 Feb 2002)
