$ulevel

$ulevel



The $ulevel identifier returns the matching user level for any triggered On_Event.

Synopsis

$ulevel

Parameters

None

Properties

None

Example

When a user speaks in a channel, echo the user level that is triggered: 
ON *:TEXT:*:#:echo -a User level triggered: $ulevel

Compatibility

Added: mIRC v4.6 (07 Sep 1996)

Notes

None
