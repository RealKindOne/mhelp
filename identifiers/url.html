$url

$url



The $url identifier returns the current url in your main browser, or returns information about URL in the URL list window.

Synopsis

$url
$url(<N>)[.properties]

Parameters

N - The Nth URL in the list or return the total number of URL if N = 0.

Properties

.desc - Returns the description of the URL
.group - Returns the group of the URL

Example

//echo -a $url

Compatibility

Added: mIRC v4.7 (09 Dec 1996)
