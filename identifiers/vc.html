$vc



The $vc identifier is used to retrieve commands in the voice command list, same as $vcmd.

Synopsis

$vc(N)

Parameters

N - Used to specify the Nth command in the list, or can be 0 to retrieve the total amount of commands in the voice command list.

Properties

None

Example

None

Compatibility

Added: mIRC v6.0 (03 Feb 2002)

None
