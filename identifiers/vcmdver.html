$vcmdver

$vcmdver



The $vcmdver identifier is used to retrieve the version of your installed Speech Recognition software.

If you do not have Speech Recognition software installed you will get $null.

Synopsis

$vcmdver

Parameters

None

Properties

None

Example

Echo the current version of your Speech Recognition software to the active window: 
//echo -a $vcmdver

Compatibility

Added: mIRC v5.7 (02 Feb 2000)

Notes

None
