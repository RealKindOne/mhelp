$vnick()



The $vnick() identifier returns the Nth voiced nick in a channel.

Note: This feature is depreciated and has been replaced with $nick().

Synopsis

$vnick(#,N)

Parameters

None

Properties

None

Example

Echo total number of voiced nicks in #foobar: 
//echo -a $vnick(#foobar,0)
Echo 1st voiced nick in #foobar: 
//echo -a $vnick(#foobar,1)

Compatibility

Added: mIRC v4.5 (30 Jun 1996)

Depreciated: mIRC v5.6 (03 Jun 1999)
