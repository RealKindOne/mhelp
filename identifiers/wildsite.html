$wildsite



The $wildsite identifier returns the address of the user who triggered an event in the form *!*@host.

Synopsis

$wildsite

Parameters

None

Properties

None

Example

on *:text:*:#:{
  echo -a $wildsite
}

Compatibility

Added: mIRC v4.7 (09 Dec 1996)

Notes

None
