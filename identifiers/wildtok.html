$wildtok



The $wildtok identifier returns the Nth token that matches the wildcard string. $wildtokcs is the case sensitive equivalent.

Synopsis

$wildtok(<list>,wildstring,N,C)

Parameters

<list> - The list of tokens (can be empty)
wildstring - a wildcard string token used to match
N - The Nth matching token, default to 1, if N is 0, returns the total number of matching tokens.
<C> - The ascii number (or code point) of the character separating the tokens

Properties

None

Example

return three 
$wildtok(one two three,t*e,1,32)

Compatibility

Added: mIRC v4.7 (09 Dec 1996)
