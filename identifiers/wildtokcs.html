$wildtokcs



The $wildtokcs identifier returns the Nth token that matches the wildcard string, using case sensitivity. $wildtok is the case insensitive equivalent.

Synopsis

$wildtokcs(<list>,wildstring,N,C)

Parameters

<list> - The list of tokens (can be empty)
wildstring - a wildcard string token used to match
N - The Nth matching token, default to 1, if N is 0, returns the total number of matching tokens. Cannot be negative. If N is greater than the number of token, $null is returned.
<C> - The ascii number (or code point) of the character separating the tokens

Properties

None

Example

Return twO: 
//echo -a $wildtokcs(one two three twO, t*O,1,32)

Compatibility

Added: mIRC v5.91 (15 Jun 2001)
