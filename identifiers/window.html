$window

$window



The $window identifier returns information about the specified window for the current connection or returns the name of the @window that is left inside the 'leave' menu {} event.

Synopsis

$window

Parameters

None

Properties

None

Example

example {
  .part:part $window
}

Added: mIRC v5.0 (02 Apr 1997)
